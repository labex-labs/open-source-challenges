# 电子商务网站

首先，打开编辑器。你可以在编辑器中看到以下文件。

```txt
├── public
├── src
│   ├── components
│   ├── context
│   ├── mockData
│   ├── App.css
│   ├── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## 要求

- 要安装项目依赖项，请使用以下命令：

  ```bash
  npm i
  ```

- 请在 `src/components/itemDetail/ItemDetail.js` 文件中完成此挑战。
- 定义了 `getItemDetail` 函数。它接受一个 id 参数，并从与给定 id 匹配的 items 数组中返回商品对象。
- 实现点击商品以显示商品详情页面的功能。

## 示例

完成代码后，使用以下命令运行它：

```bash
npm start
```
