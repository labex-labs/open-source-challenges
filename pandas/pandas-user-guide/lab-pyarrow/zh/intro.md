# 简介

本实验将指导你学习如何在 pandas 中使用 PyArrow，以扩展功能并提高各种 API 的性能。PyArrow 为 pandas 提供了更广泛的数据类型、对所有数据类型的缺失数据支持、IO 读取器集成以及与其他数据框库的互操作性。
