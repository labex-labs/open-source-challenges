# 简介

在本实验中，我们将处理空气质量数据，以探索如何使用 Python 的 Pandas 库来合并多个表格。我们将使用 `concat` 和 `merge` 函数来执行这些操作。本实验将帮助你了解如何有效地连接和合并数据框。
