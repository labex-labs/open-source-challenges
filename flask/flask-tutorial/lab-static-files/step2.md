# Add CSS rules

Next, copy the following CSS rules into the `style.css` file. These rules will style various HTML elements in our application.

```css
/* flaskr/static/style.css */

html {
  font-family: sans-serif;
  background: #eee;
  padding: 1rem;
}
body {
  max-width: 960px;
  margin: 0 auto;
  background: white;
}
/* More CSS rules... */
```
