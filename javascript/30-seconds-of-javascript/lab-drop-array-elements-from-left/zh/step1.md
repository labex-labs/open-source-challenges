# 从数组左侧移除元素

要开始练习编码，请打开终端/SSH 并输入 `node`。

以下是一个函数，它创建一个新数组，其中从左侧移除了指定数量的元素：

```js
const drop = (arr, n = 1) => arr.slice(n);
```

该函数使用 `Array.prototype.slice()` 从左侧移除指定数量的元素。如果你省略最后一个参数 `n`，该函数将使用默认值 `1`。

以下是使用 `drop` 函数的一些示例：

```js
drop([1, 2, 3]); // [2, 3]
drop([1, 2, 3], 2); // [3]
drop([1, 2, 3], 42); // []
```
