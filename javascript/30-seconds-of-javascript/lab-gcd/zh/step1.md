# 如何计算最大公约数

要使用代码计算两个或多个数字/数组之间的最大公约数，请遵循以下步骤：

1. 打开终端/SSH 并输入 `node` 开始练习编码。

2. 使用以下代码：

```js
const gcd = (...arr) => {
  const _gcd = (x, y) => (!y ? x : gcd(y, x % y));
  return [...arr].reduce((a, b) => _gcd(a, b));
};
```

3. `gcd` 函数使用递归。

4. 基线条件是当 `y` 等于 `0` 时。在这种情况下，函数返回 `x`。

5. 否则，函数返回 `y` 和除法 `x / y` 的余数的最大公约数。

6. 要测试该函数，请使用以下代码：

```js
gcd(8, 36); // 4
gcd(...[12, 8, 32]); // 4
```
