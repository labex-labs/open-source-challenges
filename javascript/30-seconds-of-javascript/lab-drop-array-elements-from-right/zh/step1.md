# 从数组右侧删除元素

要从数组右侧删除指定数量的元素，请执行以下步骤：

1. 打开终端/SSH 并输入 `node` 开始练习编码。
2. 使用 `Array.prototype.slice()` 从右侧删除指定数量的元素。
3. 如果你只想删除一个元素，可以省略最后一个参数 `n`，将使用默认值 `1`。

以下是一个示例代码片段：

```js
const dropRight = (arr, n = 1) => arr.slice(0, -n);
```

你可以使用以下示例测试此函数：

```js
dropRight([1, 2, 3]); // [1, 2]
dropRight([1, 2, 3], 2); // [1]
dropRight([1, 2, 3], 42); // []
```
