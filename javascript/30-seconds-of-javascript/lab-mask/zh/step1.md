# 如何在 JavaScript 中对值进行掩码处理

要在 JavaScript 中对值进行掩码处理，你可以使用 `mask()` 函数。请按以下步骤操作：

1. 打开终端/SSH 并输入 `node` 以开始练习编码。
2. 使用 `String.prototype.slice()` 来获取将保持未掩码状态的字符部分。
3. 使用 `String.prototype.padStart()` 用掩码字符填充字符串的开头，直至达到原始长度。
4. 如果你想从字符串末尾排除字符，请对 `num` 使用负值。
5. 如果你未为 `num` 指定值，函数将默认保留最后 4 个字符不进行掩码处理。
6. 如果你未为 `mask` 指定值，函数将默认使用 `'*'` 字符作为掩码。

以下是 `mask()` 函数的代码：

```js
const mask = (cc, num = 4, mask = "*") =>
  `${cc}`.slice(-num).padStart(`${cc}`.length, mask);
```

以下是一些使用 `mask()` 函数的示例：

```js
mask(1234567890); // '******7890'
mask(1234567890, 3); // '*******890'
mask(1234567890, -4, "$"); // '$$$$567890'
```
