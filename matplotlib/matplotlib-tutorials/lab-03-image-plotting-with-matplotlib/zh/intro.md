# 简介

在本实验中，你将学习如何使用 Python 中的 Matplotlib 库绘制和处理图像。你将学习如何将图像数据导入 NumPy 数组，将 NumPy 数组绘制成图像，应用伪彩色方案，添加颜色刻度参考，检查特定数据范围，以及探索不同的插值方案。
