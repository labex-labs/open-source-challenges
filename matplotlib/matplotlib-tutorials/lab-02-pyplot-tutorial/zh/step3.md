# 绘制多条线

我们还可以在一次函数调用中使用数组绘制多条具有不同样式的线。让我们绘制三条线：一条红色虚线、蓝色方块和绿色三角形：

```python
import numpy as np

t = np.arange(0., 5., 0.2)

plt.plot(t, t, 'r--', t, t**2, 'bs', t, t**3, 'g^')
plt.show()
```

解释：

- 我们使用 `numpy` 模块创建一个包含均匀采样时间值的数组 `t`。
- `plot` 函数被调用，带有三对 `x` 和 `y` 值，后面跟着格式字符串 `'r--'`（红色虚线）、`'bs'`（蓝色方块）和 `'g^'`（绿色三角形）。
