# 简介

在本实验中，你将学习如何使用 Python 中的 Matplotlib 创建一个简单的图形用户界面（GUI）。具体来说，你将创建一个傅里叶演示，在频域和时域中显示两个波形。你可以通过在图上点击并拖动来调整波形的频率和幅度。
