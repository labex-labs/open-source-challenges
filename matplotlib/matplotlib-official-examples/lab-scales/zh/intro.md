# 简介

Matplotlib 是一个用于数据可视化的 Python 库。它允许用户创建各种各样的图表、绘图和图形。Matplotlib 的一个关键特性是它能够对坐标轴应用比例变换。这在呈现数据时提供了更大的灵活性，特别是在处理非常大或非常小的数字时。在本实验中，我们将学习如何使用 Matplotlib 对坐标轴应用各种比例变换。
