# 简介

在本实验中，你将学习如何使用 Python 的 Matplotlib 库创建等高线图像。等高线图像是一种可视化技术，它使用二维绘图来表示三维表面。等高线图像由等高线组成，等高线是连接表面上等值点的线，以及填充等高线，即等高线之间的区域。
