# 定义等高线和多边形

下一步是定义等高线和多边形。在这个例子中，我们有两个级别之间的线条和填充等高线。

```python
# 每个级别的等高线是多边形的列表/元组。
lines0 = [[[0, 0], [0, 4]]]
lines1 = [[[2, 0], [1, 2], [1, 3]]]
lines2 = [[[3, 0], [3, 2]], [[3, 3], [3, 4]]]  # 注意有两条线。

# 两个级别之间的填充等高线也是多边形的列表/元组。
# 点可以按顺时针或逆时针顺序排列。
filled01 = [[[0, 0], [0, 4], [1, 3], [1, 2], [2, 0]]]
filled12 = [[[2, 0], [3, 0], [3, 2], [1, 3], [1, 2]],   # 注意有两个多边形。
            [[1, 4], [3, 4], [3, 3]]]
```
