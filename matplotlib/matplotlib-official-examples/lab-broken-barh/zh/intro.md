# 简介

Matplotlib 是一个用于在 Python 中创建静态、动画和交互式可视化效果的 Python 库。它是 Python 中用于数据可视化的一个流行库。在本教程中，我们将学习如何使用 Matplotlib 创建一个水平条形断裂图。
