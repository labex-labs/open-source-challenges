# 简介

在数据可视化中，放大图表的特定区域对于更好地理解和分析数据非常有用。Matplotlib 是一个广受欢迎的 Python 数据可视化库，它提供了一种方法来创建两个相同的面板，并在右侧面板中放大，以在第一个面板中显示一个矩形，表示缩放区域。在本实验中，我们将学习如何在 Matplotlib 中创建此交互式缩放功能。
