# 简介

在本实验中，你将学习如何使用 Python 中的 Matplotlib 创建三维等高线图。等高线图是三个变量之间关系的图形表示。它用于在 x 轴和 y 轴上显示两个变量之间的关系，以及在 z 轴上显示第三个变量。等高线图在科学和工程领域中广泛用于在三维空间中显示数据。
