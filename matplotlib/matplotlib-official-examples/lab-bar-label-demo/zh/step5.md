# 使用 `{}` 风格格式字符串进行柱状图标注

在这一步中，我们将展示如何使用 `{}` 风格的格式字符串来格式化柱状图标签。我们将使用一些按口味分类的意式冰淇淋销售数据。

```python
水果名称 = ['咖啡味', '咸焦糖味', '开心果味']
水果销量 = [4000, 2000, 7000]

fig, ax = plt.subplots()
柱状图容器 = ax.bar(水果名称, 水果销量)
ax.set(ylabel='销售的品脱数', title='按口味分类的意式冰淇淋销售情况', ylim=(0, 8000))
ax.bar_label(柱状图容器, fmt='{:,.0f}')
```
