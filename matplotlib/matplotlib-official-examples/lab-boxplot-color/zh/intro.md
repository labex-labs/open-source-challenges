# 简介

本教程将指导你使用 Python Matplotlib 为箱线图创建自定义填充颜色的过程。箱线图是一种用于展示一组数据分布的图表类型。它们展示了数据集的中位数、四分位数和异常值。在本教程中，我们将使用 Matplotlib 中的 `boxplot()` 函数创建两种类型的箱线图（矩形和带缺口的），并用自定义颜色填充它们。
