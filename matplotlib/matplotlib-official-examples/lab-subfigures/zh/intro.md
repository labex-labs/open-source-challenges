# 简介

Matplotlib 是一个用于创建可视化图表（如图表、图形等）的 Python 库。在本实验中，你将学习如何在 Matplotlib 中创建子图。子图是指在同一图形中包含两个或更多个绘图的图形。子图可以帮助你比较不同的数据集，或者展示同一数据的不同视图。当你想要一起显示多个绘图时，它们非常有用。
