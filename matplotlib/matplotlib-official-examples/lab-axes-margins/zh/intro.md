# 简介

Matplotlib 是 Python 中的一个数据可视化库，用于在 Python 中创建静态、动画和交互式可视化。它提供了一个面向对象的 API，用于使用 Tkinter、wxPython、Qt 或 GTK 等通用 GUI 工具包将绘图嵌入应用程序中。在本实验中，我们将学习如何使用 Python 在 Matplotlib 中控制视图限制和粘性边缘。
