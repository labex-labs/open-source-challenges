# 简介

Matplotlib 是一个用于 Python 编程语言及其数值数学扩展 NumPy 的绘图库。Matplotlib 的特性之一是能够创建自定义的框样式。

在本实验中，你将学习如何在 Matplotlib 中实现自定义框样式。你将学习如何将自定义框样式创建为函数和类。你还将学习如何向 Matplotlib 注册自定义框样式。
