# 简介

本教程将指导你使用 Python 中的 Matplotlib 库绘制功率谱密度（PSD）的过程。PSD 是信号处理领域常用的一种图表。NumPy 有许多用于计算 PSD 的有用库，我们将展示一些如何使用 Matplotlib 完成并可视化 PSD 的示例。
