# 简介

本实验教程将指导你在Flask Web应用服务器中使用Matplotlib。你将学习如何使用`.Figure`构造函数创建图形并将其保存到内存缓冲区，将生成的图形嵌入HTML输出中，以及使用`flask`命令行工具运行Flask应用程序。
