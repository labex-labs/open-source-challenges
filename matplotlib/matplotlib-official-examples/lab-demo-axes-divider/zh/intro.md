# 简介

Matplotlib 是一个 Python 库，可用于在 Python 中创建静态、动画和交互式可视化。它广泛应用于科学计算、数据分析、机器学习等领域。在本实验中，你将学习如何使用 Matplotlib 绘制图像，以及如何操作坐标轴和颜色条的位置。
