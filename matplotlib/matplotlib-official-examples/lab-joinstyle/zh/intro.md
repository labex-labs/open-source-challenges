# 简介

本实验提供了一份关于如何在 Python 中使用 Matplotlib 的 `JoinStyle` 模块的详细指南。`JoinStyle` 控制着 Matplotlib 绘制两条不同线段相交处的角的方式。本实验是为刚接触 Matplotlib 的用户设计的。
