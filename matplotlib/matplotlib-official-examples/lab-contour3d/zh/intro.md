# 简介

在本实验中，我们将学习如何使用 Python 中的 Matplotlib 库创建三维等高线图。等高线图是三维表面的图形表示，其中等高线绘制在二维平面上。等高线图有助于可视化一个变量相对于其他两个变量的变化。
