# 简介

Matplotlib 是 Python 中一个强大的数据可视化库。它提供了各种图表来以有意义的方式可视化数据。在本实验中，我们将学习如何使用 Matplotlib 中的 `fig.add_axes` 在主绘图轴内创建内嵌轴。
