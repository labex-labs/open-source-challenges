# 简介

Matplotlib 是 Python 中一个流行的数据可视化库。它通过 FreeType 库提供字体支持。在本实验中，我们将学习如何使用 Matplotlib 的 `Axes.table` 绘制给定字体的前 255 个字符的字体表。
