# 简介

本实验将指导你使用 Python 中的 Matplotlib 库创建 3D 体素图的过程。体素图对于以清晰且美观的方式可视化 3D 数据非常有用。在本实验中，我们将使用 `Axes3D.voxels` 函数创建两个立方体及其之间连接的体素图。
