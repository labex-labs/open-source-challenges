# 简介

Matplotlib 是 Python 中广泛使用的数据可视化库。它提供了各种工具来创建不同类型的图表，包括子图。在创建子图时，给每个图表添加标签通常有助于读者更轻松地理解所呈现的信息。在本实验中，我们将学习如何使用 Matplotlib 提供的不同方法给子图添加标签。
