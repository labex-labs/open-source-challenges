# 简介

本实验将指导你使用 Python 和 Matplotlib 创建一个 3D 盒状表面图。我们将定义尺寸、创建虚拟数据并绘制等高面。我们将设置绘图的限制、绘制边缘、设置标签和 z 轴刻度、设置缩放和视角，并添加颜色条。
