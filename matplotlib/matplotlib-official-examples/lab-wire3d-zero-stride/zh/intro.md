# 简介

本实验旨在向你展示如何使用 Matplotlib 在 Python 中创建 3D 线框绘图。线框绘图是三维曲面的可视化表示，它使用线条来显示曲面的结构。在本实验中，我们将展示如何设置 `rstride` 和 `cstride` 参数来控制线图中线条的密度。
