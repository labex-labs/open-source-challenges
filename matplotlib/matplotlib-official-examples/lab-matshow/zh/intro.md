# 简介

Matplotlib 是 Python 中一个流行的数据可视化库。它可用于创建各种不同类型的图表。Matplotlib 提供的功能之一是 `matshow()`，它可用于将二维矩阵或数组可视化为彩色编码图像。在本实验中，我们将逐步介绍使用 `matshow()` 可视化二维数组的步骤。
