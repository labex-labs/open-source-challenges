# 简介

Matplotlib 是一个广受欢迎的 Python 库，用于在 Python 中创建静态、动画和交互式可视化。在本实验中，你将学习如何将 Matplotlib 嵌入到 GTK3 应用程序中，并与树状视图进行交互以存储数据。
