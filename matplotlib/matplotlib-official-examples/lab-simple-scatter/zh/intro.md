# 简介

本实验旨在教你如何使用 Python 的 Matplotlib 库创建动画散点图。我们将涵盖从设置图表到将动画保存为 GIF 的所有内容。在本实验结束时，你将拥有一个可用的动画散点图，可用于可视化你的数据。
