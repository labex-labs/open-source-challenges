# 简介

在本实验中，我们将学习如何使用 Python 的 Matplotlib 库创建水平条形图。我们将以体育老师想要向家长展示他们的孩子在一些体能测试中的表现与其他孩子相比如何为例。为了演示目的，我们将为小约翰尼·多伊编造一些数据来提取绘图代码。
