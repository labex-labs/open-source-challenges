# 简介

在本实验中，我们将学习如何使用 Python 中的 Matplotlib 库创建 SkewT-logP 图。SkewT-logP 图常用于气象学中显示温度的垂直剖面。它是一个复杂的图表，因为它涉及非正交的 X 和 Y 轴。我们将使用 Matplotlib 的变换和自定义投影 API 来创建此图表。
