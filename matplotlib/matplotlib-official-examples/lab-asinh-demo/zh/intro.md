# 简介

Matplotlib 是 Python 中一个强大的数据可视化库。它提供了各种工具来创建各种各样的图形、图表和绘图。Matplotlib 最强大的功能之一是其缩放数据的能力。本实验将向你介绍反双曲正弦缩放（AsinhScale），它是一种变换，允许绘制涵盖非常宽动态范围（包括正值和负值）的数量。
