# 简介

本实验是关于使用 Python Matplotlib 库设置颜色的透明度值。透明度值是透明度的一种度量，其中 0 值表示完全透明，1 值表示完全不透明。我们将探索在 Matplotlib 中设置透明度值的两种方法：使用 `alpha` 关键字参数和使用 `(matplotlib_color, alpha)` 颜色格式。
