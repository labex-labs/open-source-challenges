# 简介

在本实验中，你将学习如何使用标记的不同属性，通过 Python 的 Matplotlib 库绘制多元数据集。具体来说，你将学习如何将一次成功的棒球投掷表示为一个笑脸，其中标记大小映射到投掷者的技能，标记旋转映射到起飞角度，推力映射到标记颜色。
