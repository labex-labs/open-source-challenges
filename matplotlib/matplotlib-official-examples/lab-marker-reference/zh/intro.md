# 简介

Matplotlib 是一个广受欢迎的 Python 库，用于创建可视化效果，包括图表、图形和绘图。Matplotlib 的关键组件之一是标记（marker），用于在绘图上表示数据点。标记有各种形状、大小和样式，可以进行定制以适应特定的数据集。在本实验中，你将学习如何使用 Matplotlib 标记来创建能够有效传达数据的自定义可视化效果。
