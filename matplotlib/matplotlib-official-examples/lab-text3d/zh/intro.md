# 简介

本实验将演示如何使用 Python 中的 Matplotlib 库在 3D 图上放置文本注释。将涵盖以下功能：

- 使用 `~.Axes3D.text` 函数，并设置三种类型的 `zdir` 值：None、轴名称（例如 'x'）或方向元组（例如 (1, 1, 0)）。
- 使用 `~.Axes3D.text` 函数，并设置颜色关键字。
- 使用 `.text2D` 函数在 `ax` 对象的固定位置放置文本。
