# 简介

在本实验中，我们将学习如何使用 Python 的 Matplotlib 库创建一个简单的散点图。散点图是一种将两个变量的值显示为点的集合的图表类型。每个点代表两个变量的值，点的位置表示这两个变量的值。散点图对于识别变量之间的关系和识别异常值很有用。
