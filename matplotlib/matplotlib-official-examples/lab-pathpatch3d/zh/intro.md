# 简介

Matplotlib 是 Python 中一个流行的数据可视化库。它提供了广泛的函数来创建不同类型的绘图和图表。Matplotlib 的特性之一是能够在 3D 绘图中绘制平面物体。本实验将指导你完成使用 Matplotlib 在 3D 绘图中绘制平面物体的过程。
