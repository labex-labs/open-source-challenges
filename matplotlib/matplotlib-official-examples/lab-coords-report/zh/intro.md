# 简介

本逐步实验将指导你使用 Python 的 Matplotlib 库创建可视化图表。Matplotlib 是一个数据可视化库，允许用户创建各种可视化图表，包括折线图、散点图和直方图。在本实验中，我们将使用 Matplotlib 创建一个简单的散点图。
