# 简介

本实验将指导你直接使用 agg 后端在 Python Matplotlib 中创建图像的过程。agg 后端对于那些希望在不使用 pyplot 接口来管理图形、关闭图形等情况下完全控制代码的 Web 应用程序开发人员很有用。在本实验中，我们将向你展示如何将 agg 画布的内容保存到文件以及如何将其提取到 numpy 数组中，该数组又可以传递给 Pillow。
