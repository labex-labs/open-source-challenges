# 简介

在本实验中，我们将学习如何使用 Python 中的 Matplotlib 创建日期图。我们将使用 `matplotlib.dates` 模块将 datetime 对象转换为 Matplotlib 的内部表示形式。我们还将学习如何格式化 x 轴上的刻度标签，以便以可读格式显示日期。
