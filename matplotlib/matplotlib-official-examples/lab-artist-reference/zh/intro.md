# 简介

在本实验中，我们将学习如何使用 Python 中的 Matplotlib 库绘制各种形状。Matplotlib 是一个用于 Python 编程语言及其数值数学扩展 NumPy 的绘图库。它提供了一个面向对象的 API，用于使用 Tkinter、wxPython、Qt 或 GTK 等通用 GUI 工具包将绘图嵌入到应用程序中。
