# 简介

在本实验中，我们将学习如何使用 `transforms.offset_copy` 将绘图元素（如文本字符串）相对于以任何坐标给出的位置，在屏幕坐标中的指定偏移量处定位。我们将使用 Python 中的 Matplotlib 库创建一个散点图和一个极坐标图，然后使用 `offset_copy` 向每个图中添加文本。
