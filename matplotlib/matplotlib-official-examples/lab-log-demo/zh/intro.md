# 简介

本逐步教程将指导你使用 Python Matplotlib 创建带有对数轴的图表。本教程将涵盖以下主题：

1. 半对数 y 轴图（Semilogy Plot）
2. 半对数 x 轴图（Semilogx Plot）
3. 双对数轴图（Loglog Plot）
4. 误差线图（Errorbars Plot）
