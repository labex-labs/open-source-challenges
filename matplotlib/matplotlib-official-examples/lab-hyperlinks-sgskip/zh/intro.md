# 简介

Matplotlib 是一个 Python 数据可视化库，它能让用户在 Python 中创建各种各样的静态、动画和交互式可视化图表。在本实验中，你将学习如何使用 Matplotlib 为你的图表添加超链接。在本实验结束时，你将能够为 Matplotlib 中的散点图和图像添加超链接。
