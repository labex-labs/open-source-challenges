# 简介

在数据可视化中，等高线图通常用于在二维平面上显示三维数据。Matplotlib 是 Python 中广泛使用的绘图库，它提供了创建不同类型图表的功能，包括等高线图。在本实验中，我们将学习如何使用 Matplotlib 创建掩码等高线图，以及如何说明启用和禁用角掩码之间的区别。
