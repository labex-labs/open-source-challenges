# 简介

Matplotlib 是一个广受欢迎的用于 Python 的数据可视化库。本教程将指导你创建一个带状箱形图，这是一种可视化数据的独特方式。带状箱形图是一种堆叠条形图，其中每个条形都是“带状”的，并且具有渐变配色方案。
