# 简介

Matplotlib 是 Python 中的一个数据可视化库，它提供了各种工具和技术来创建视觉上吸引人的图表。有时，需要为 y 轴标签留出空间，以避免文本重叠或截断。本实验将向你展示如何使用 axes_grid 模块在 Matplotlib 中为 y 轴标签留出空间。
