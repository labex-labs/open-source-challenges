# 简介

Matplotlib 是一个用于 Python 编程语言的数据可视化库。它提供了广泛的工具来创建不同类型的绘图和图表。Matplotlib 中的 GridSpec 模块允许我们创建灵活且复杂的子图布局。在本教程中，我们将学习如何使用 GridSpec 创建多列/行子图布局。
