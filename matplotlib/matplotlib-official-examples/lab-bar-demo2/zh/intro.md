# 简介

本教程将指导你逐步完成使用 Python 的 Matplotlib 库创建柱状图的过程。本教程中的示例展示了如何使用厘米和英寸的默认单位，如何使用各种关键字设置 x 和 y 单位，以及如何使用标量或单位设置 x 轴范围。
