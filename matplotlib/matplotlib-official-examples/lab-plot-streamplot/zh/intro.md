# 简介

本教程将指导你使用 Python 中的 Matplotlib 创建流线图的过程。流线图是一种二维向量场，用于显示一组流线。它用于可视化流体流动和其他向量场。在本教程中，我们将向你展示如何创建一个具有变化密度、颜色和线宽的流线图，同时控制流线的起点。
