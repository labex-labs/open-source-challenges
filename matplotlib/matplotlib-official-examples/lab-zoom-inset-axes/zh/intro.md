# 简介

Matplotlib 是一个用于在 Python 中创建静态、动画和交互式可视化的 Python 库。它是数据可视化的强大工具，在科学界被广泛使用。在本实验中，我们将学习如何在 Matplotlib 绘图中创建一个放大区域。
