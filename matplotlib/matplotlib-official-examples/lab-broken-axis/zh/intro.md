# 简介

在数据可视化中，有时我们需要处理离群值，这些离群值会让我们难以看清大部分数据的细节。在这种情况下，我们可以使用断轴（broken axis）来放大显示大部分数据，同时仍展示离群值。在本教程中，你将学习如何使用 Python 中的 Matplotlib 创建断轴图。
