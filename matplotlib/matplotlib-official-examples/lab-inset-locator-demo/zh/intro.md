# 简介

Matplotlib 是 Python 中的一个数据可视化库。`.inset_locator` 功能允许在坐标轴的角落轻松放置插图。在本实验中，我们将探索如何使用 `.inset_locator` 功能在 Matplotlib 绘图中创建插图。
