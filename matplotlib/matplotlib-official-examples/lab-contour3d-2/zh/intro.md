# 简介

本教程将指导你使用 Python 中的 Matplotlib 创建三维等高线图的过程。等高线图使用等高线或等值线来表示三维表面。我们将使用 `contour()` 函数来创建这些等值线，并使用 `extend3d=True` 选项将曲线垂直扩展为“带状”。
