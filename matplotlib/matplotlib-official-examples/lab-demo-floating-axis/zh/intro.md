# 简介

Matplotlib 是一个用于在 Python 中创建静态、动画和交互式可视化效果的 Python 库。Matplotlib 的一个关键特性是它能够创建各种类型和样式的二维和三维图形，包括散点图、折线图和柱状图。在本实验中，你将学习如何使用 Matplotlib 在矩形框中创建极坐标曲线。
