# 简介

Matplotlib 是一个广受欢迎的数据可视化库，它提供了大量用于在 Python 中创建可视化的工具。Matplotlib 的一个有趣特性是能够在图表上添加十字准线光标。在本实验中，你将学习如何在 Matplotlib 图表上添加十字准线光标。
