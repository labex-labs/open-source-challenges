# Summary

In summary, we have shown you how to write a program that can extract the company name from an email address in the "username@companyname.com" format. By using the `input()` function to get the email address, and `re.findall()` to find all substrings using regex, we were able to extract the company name from the email address. This program can be useful for anyone who needs to quickly extract the company name from an email address.
