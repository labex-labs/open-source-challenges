# Unicode String

Print a unicode string "hello world".

## Preparation

Before we start writing the code, we should open the `/home/labex/project/unicode_string.py` file in WebIDE.

## Start writing code

Then input the following code.

```python
unicodeString = u"hello world!"
print(unicodeString)
```

This Python code defines a Unicode string variable called `unicodeString` and assigns it the value `"hello world!"`. The `u` prefix before the string indicates that it is a Unicode string.

The code then uses the `print()` function to output the value of the `unicodeString` variable to the console.

Overall, this code demonstrates how to define and print a Unicode string in Python using the `u` prefix.
