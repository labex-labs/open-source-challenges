# 私有属性

任何以单下划线 `_` 开头的属性名都被视为**私有**属性。

```python
class Person(object):
    def __init__(self, name):
        self._name = 0
```

如前所述，这只是一种编程风格。你仍然可以访问和更改它。

```python
>>> p = Person('Guido')
>>> p._name
'Guido'
>>> p._name = 'Dave'
>>>
```

一般来说，任何以单下划线 `_` 开头的名称，无论是变量、函数还是模块名，都被视为内部实现。如果你发现自己直接使用这样的名称，那你可能做错了。应该寻找更高级别的功能。
