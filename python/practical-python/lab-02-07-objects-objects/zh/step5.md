# 身份标识与引用

使用 `is` 运算符来检查两个值是否为同一个对象。

```python
>>> a = [1,2,3]
>>> b = a
>>> a is b
True
>>>
```

`is` 比较的是对象的身份标识（一个整数）。可以使用 `id()` 函数获取对象的身份标识。

```python
>>> id(a)
3588944
>>> id(b)
3588944
>>>
```

注意：检查对象时，几乎总是使用 `==` 更好。`is` 的行为常常出人意料：

```python
>>> a = [1,2,3]
>>> b = a
>>> c = [1,2,3]
>>> a is b
True
>>> a is c
False
>>> a == c
True
>>>
```
