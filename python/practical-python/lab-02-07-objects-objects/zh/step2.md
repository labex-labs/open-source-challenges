# 赋值示例

考虑以下代码片段。

```python
a = [1,2,3]
b = a
c = [a,b]
```

下面是底层内存操作的示意图。在这个例子中，只有一个列表对象 `[1,2,3]`，但有四个不同的引用指向它。

![内存引用示意图示例](../assets/references.png)

这意味着修改一个值会影响 _所有_ 引用。

```python
>>> a.append(999)
>>> a
[1,2,3,999]
>>> b
[1,2,3,999]
>>> c
[[1,2,3,999], [1,2,3,999]]
>>>
```

注意原始列表的更改是如何在其他所有地方都体现出来的（哎呀！）。这是因为从未进行过复制操作。所有引用都指向同一个对象。
