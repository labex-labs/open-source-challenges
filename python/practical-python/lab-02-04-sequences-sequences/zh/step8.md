# 遍历整数

如果你需要进行计数，请使用 `range()`。

```python
for i in range(100):
    # i = 0,1,...,99
```

其语法为 `range([start,] end [,step])`

```python
for i in range(100):
    # i = 0,1,...,99
for j in range(10,20):
    # j = 10,11,..., 19
for k in range(10,50,2):
    # k = 10,12,...,48
    # 注意它是以 2 为步长计数，而非 1。
```

- 结束值永远不包含在内。这与切片的行为一致。
- `start` 是可选的。默认值为 `0`。
- `step` 是可选的。默认值为 `1`。
- `range()` 根据需要计算值。它实际上并不存储大量数字。
