# 整数（int）

具有任意大小和基数的有符号值：

```python
a = 37
b = -299392993727716627377128481812241231
c = 0x7fa8      # 十六进制
d = 0o253       # 八进制
e = 0b10001111  # 二进制
```

常见操作：

    x + y      加法
    x - y      减法
    x * y      乘法
    x / y      除法（产生浮点数）
    x // y     整除（产生整数）
    x % y      取模（余数）
    x ** y     幂运算
    x << n     按位左移
    x >> n     按位右移
    x & y      按位与
    x | y      按位或
    x ^ y      按位异或
    ~x         按位取反
    abs(x)     绝对值
