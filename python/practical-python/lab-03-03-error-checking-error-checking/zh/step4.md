# 内置异常

大约有二十几种内置异常。通常，异常的名称能表明出了什么问题（例如，引发 `ValueError` 是因为你提供了一个错误的值）。这并非完整列表。如需更多信息，请查看[文档](https://docs.python.org/3/library/exceptions.html)。

```python
ArithmeticError
AssertionError
EnvironmentError
EOFError
ImportError
IndexError
KeyboardInterrupt
KeyError
MemoryError
NameError
ReferenceError
RuntimeError
SyntaxError
SystemError
TypeError
ValueError
```
