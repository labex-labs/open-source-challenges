# 字典/类的增长

Python 字典（以及类）在其预留内存翻倍之前允许存储多达 5 个值。通过创建一个字典并向其中添加更多值来进行研究：

```python
>>> row = { 'route': '22', 'date': '01/01/2001', 'daytype': 'U', 'rides': 7354 }
>>> sys.getsizeof(row)
>>> sys.getsizeof(row)
240
>>> row['a'] = 1
>>> sys.getsizeof(row)
240
>>> row['b'] = 2
>>> sys.getsizeof(row)
368
>>>
```

如果你删除刚添加的项，内存会减少吗？

思考要点：如果你要创建大量记录，将每个记录表示为字典可能不是最有效的方法——你可能会为使用字典的便利性付出沉重代价。或许最好考虑使用元组、具名元组或定义了 `__slots__` 的类。
