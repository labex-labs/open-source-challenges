# 简介

**目标**：

- 定义模块
- 使用 `import` 语句

**注意**：对于这个涉及模块的练习，确保你在合适的环境中运行 Python 至关重要。如果你无法使 `import` 语句生效，可能需要检查 `sys.path` 的值。如果一切看起来都有问题，请寻求帮助。

在开始这个练习之前，首先重启你的 Python 解释器会话。如果使用 IDLE，点击 shell 窗口并查找菜单选项“Shell \> Restart Shell”。你应该会看到这样的消息：

```python
>>> ##################== RESTART ##################==
>>>
```

如果你使用的是 Unix，只需退出 Python 并重新启动解释器。
