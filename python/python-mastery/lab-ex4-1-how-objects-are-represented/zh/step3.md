# 实例数据的修改

尝试在上述其中一个实例上设置一个新属性：

```python
>>> goog.date = "6/11/2007"
>>> goog.__dict__
... 查看输出...
>>> ibm.__dict__
... 查看输出...
>>>
```

在上述输出中，你会注意到 `goog` 实例有一个 `date` 属性，而 `ibm` 实例没有。需要注意的是，Python 实际上对属性并没有任何限制。例如，实例的属性并不局限于在 `__init__()` 方法中设置的那些。

与其设置一个属性，不如尝试直接在 `__dict__` 对象中放入一个新值：

```python
>>> goog.__dict__['time'] = '9:45am'
>>> goog.time
'9:45am'
>>>
```

在这里，你真正注意到了实例是字典之上的一层这个事实。
