# 捕获异常

不要在遇到错误数据时崩溃，而是修改代码以发出警告消息。最终结果应该是成功转换的行的列表。例如：

```python
>>> port = read_csv_as_dicts('missing.csv', types=[str, int, float])
第4行：错误的行：['C', '', '53.08']
第7行：错误的行：['DIS', '50', 'N/A']
第8行：错误的行：['GE', '', '37.23']
第13行：错误的行：['INTC', '', '21.84']
第17行：错误的行：['MCD', '', '51.11']
第19行：错误的行：['MO', '', '70.09']
第22行：错误的行：['PFE', '', '26.40']
第26行：错误的行：['VZ', '', '42.92']
>>> len(port)
20
>>>
```

注意：由于你之前使用了内置函数`map()`，进行此更改可能会有点棘手。你可能不得不放弃这种方法，因为在`map()`中没有简单的方法来捕获和处理异常。
