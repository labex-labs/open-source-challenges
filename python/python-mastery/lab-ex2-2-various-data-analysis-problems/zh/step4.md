# 数据分析挑战

在上一个实验中，你编写了一些代码来读取与芝加哥交通管理局相关的CSV数据。例如，你可以像这样将数据作为字典获取：

```python
>>> import readrides
>>> rows = readrides.read_rides_as_dicts('/home/labex/project/ctabus.csv')
>>>
```

如果做了所有这些工作却不对数据进行任何处理，那就太可惜了。

在本实验中，你的任务是：编写一个程序来回答以下四个问题：

1. 芝加哥有多少条公交线路？

2. 2011年2月2日有多少人乘坐了22路公交车？你选择的任何日期的任何线路呢？

3. 每条公交线路的总乘坐人数是多少？

4. 从2001年到2011年，哪五条公交线路的乘客人数增长最多？

只要是Python标准库的一部分（即内置数据类型、标准库模块等），你可以自由使用任何技术来回答上述问题。
