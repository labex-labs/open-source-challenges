# 添加错误处理

在编写处理数据的程序时，经常会遇到与错误数据（格式错误、字段缺失等）相关的错误。修改你的`pcost.py`程序，使其读取数据文件`portfolio3.dat`并运行它（提示：它应该会崩溃）。

对你的函数稍作修改，使其能够从包含错误数据的行中恢复。例如，如果转换函数`int()`和`float()`无法转换输入，它们会引发`ValueError`异常。使用`try`和`except`来捕获并打印关于无法解析的行的警告消息。例如：

```shell
无法解析: 'C - 53.08\n'
原因: 基数为10的int()的无效文字: '-'
无法解析: 'DIS - 34.20\n'
原因: 基数为10的int()的无效文字: '-'
...
```

再次尝试在`portfolio3.dat`文件上运行你的程序。尽管会打印警告消息，但它应该能成功运行。
