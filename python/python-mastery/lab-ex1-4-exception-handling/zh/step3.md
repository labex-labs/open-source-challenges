# 交互式实验

运行你的`pcost.py`程序，并直接从交互式解释器中调用`portfolio_cost()`函数。

```python
>>> portfolio_cost('/home/labex/project/portfolio.dat')
44671.15
>>> portfolio_cost('/home/labex/project/portfolio2.dat')
19908.75
>>>
```

注意：要做到这一点，你可能需要使用`-i`选项来运行Python。例如：

```bash
python3 -i pcost.py
```

我们将会编写很多定义函数并进行交互式实验的程序。确保你知道如何做到这一点。
