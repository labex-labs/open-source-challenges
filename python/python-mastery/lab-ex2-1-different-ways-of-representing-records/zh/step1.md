# 被困在公交车上

文件 `ctabus.csv` 是一个CSV文件，包含了2001年1月1日至2013年8月31日期间芝加哥运输管理局（CTA）公交系统的每日乘客数据。它包含大约577000行数据。使用Python查看几行数据，看看它是什么样子的：

```python
>>> f = open('/home/labex/project/ctabus.csv')
>>> next(f)
'route,date,daytype,rides\n'
>>> next(f)
'3,01/01/2001,U,7354\n'
>>> next(f)
'4,01/01/2001,U,9288\n'
>>>
```

有4列数据。

- route：第0列。公交线路名称。
- date：第1列。格式为MM/DD/YYYY的日期字符串。
- daytype：第2列。日类型代码（U = 星期日/节假日，A = 星期六，W = 工作日）
- rides：第3列。乘客总数（整数）

`rides` 列记录了在给定日期乘坐该线路公交车的总人数。因此，从示例中可以看出，2001年1月1日有7354人乘坐了3路公交车。
