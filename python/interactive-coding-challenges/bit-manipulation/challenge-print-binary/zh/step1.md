# 打印二进制数

## 问题

编写一个 Python 函数，该函数接受一个介于 0 和 1 之间的实数作为输入，并返回其作为字符串的二进制表示形式。如果表示形式的长度大于 32，则返回 'ERROR'。

## 要求

为了解决这个问题，我们需要确保满足以下要求：

- 输入必须是一个浮点数。
- 输出必须是一个字符串。
- 输入的范围在 0 和 1 之间，但不包括 0 和 1 这两个值。
- 结果必须包括一个尾随零和小数点。
- 前导零和小数点计入 32 个字符的限制。
- 我们不能假定输入是有效的。
- 我们可以假定程序能装入内存。

## 示例用法

以下是该函数应有的行为示例：

- 无 -> 'ERROR'
- 超出范围 (0, 1) -> 'ERROR'
- 0.625 -> 0.101
- 0.987654321 -> 'ERROR'
