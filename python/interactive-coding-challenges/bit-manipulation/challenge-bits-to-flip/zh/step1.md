# 需要翻转的位

## 问题

给定两个整数 `A` 和 `B`，确定将 `A` 转换为 `B` 需要翻转的位数。例如，如果 `A = 11101` 且 `B = 01111`，则需要翻转的位数是 `2`。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- `A` 和 `B` 始终是整数。
- 输出始终是整数。
- `A` 和 `B` 的位数始终相同。
- 输入可能无效（`None`）。
- 解决方案应适合内存。

## 示例用法

以下是一些如何使用此函数的示例：

- 如果 `A` 或 `B` 为 `None`，则应引发异常。
- 如果 `A = 11101` 且 `B = 01111`，则需要翻转的位数是 `2`。
