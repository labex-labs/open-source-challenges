# 将 M 插入 N 中

## 问题

给定两个 16 位数字 `n` 和 `m`，以及两个索引 `i` 和 `j`，将 `m` 插入 `n` 中，使得 `m` 从第 `j` 位开始，到第 `i` 位结束。程序应处理以下情况：

- 如果没有输入任何内容，应引发异常。
- 如果为 `i` 或 `j` 提供了负索引，应引发异常。
- 如果输入无效，应引发异常。
- 如果从 `i` 到 `j` 没有足够的空间容纳 `m`，应引发异常。

程序应返回插入后的 16 位数字。

## 要求

程序应满足以下要求：

- `j` 应大于 `i`。
- 从 `i` 到 `j` 应有足够的空间容纳 `m`。
- 输入应有效。
- 程序应能装入内存。

## 示例用法

以下是程序用法的示例：

```txt
i      = 2
j      = 6
n      = 0000 0100 0000 0000
m      = 0000 0000 0001 0011
result = 0000 0100 0100 1100
```

在此示例中，`m` 被插入到 `n` 中，使得 `m` 从第 `j = 6` 位开始，到第 `i = 2` 位结束。得到的 16 位数字是 `0000 0100 0100 1100`。
