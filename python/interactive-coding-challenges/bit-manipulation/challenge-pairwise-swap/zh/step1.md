# 两两交换

## 问题

给定一个正整数，用尽可能少的操作交换其奇数位和偶数位。例如，如果输入是 `1001 1111 0110`，则输出应为 `0110 1111 1001`。

## 要求

必须满足以下要求：

- 输入始终是一个正整数。
- 程序处理 32 位。
- 输出是一个整数。
- 输入有效（不为 None）。
- 程序符合内存要求。

## 示例用法

以下示例说明了该程序的用法：

- 无 -> 异常
- 0 -> 0
- -1 -> -1
- 一般情况：

```
    输入  = 1001 1111 0110
    结果 = 0110 1111 1001
```
