# 绘制直线

## 问题

实现方法 `draw_line(screen, width, x1, x2)`，其中 `screen` 是一个字节列表，`width` 能被 8 整除，`x1` 和 `x2` 是绝对像素位置。该方法应设置 `screen` 中的相应位，以绘制从 `x1` 到 `x2` 的直线。

### 要求

`draw_line` 的实现必须满足以下要求：

- 不能假定输入是有效的。
- 必须设置 `screen` 中的相应位来绘制直线。
- 可以假定实现适合内存。

## 示例用法

以下示例说明了 `draw_line` 的预期行为：

- 无效输入 -> 异常
  - `screen` 为空
  - `width` = 0
  - 任何输入参数为 `None`
  - `x1` 或 `x2` 超出边界
- 一般情况，`len(screen)` = 20，`width` = 32：
  - `x1` = 2，`x2` = 6
    - `screen[0]` = `int('00111110', base=2)`
  - `x1` = 68，`x2` = 80
    - `screen[8]`，`int('00001111', base=2)`
    - `screen[9]`，`int('11111111', base=2)`
    - `screen[10]`，`int('10000000', base=2)`
