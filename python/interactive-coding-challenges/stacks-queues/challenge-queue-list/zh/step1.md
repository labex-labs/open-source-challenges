# 队列链表

## 问题

使用链表实现一个具有入队（enqueue）和出队（dequeue）方法的队列。入队方法应将一个元素添加到队列的末尾，而出队方法应从队列的前端移除一个元素。如果队列为空，出队操作应返回 `None`。

## 要求

为了实现该队列，我们需要遵循以下要求：

- 如果链表中有一个元素，第一个和最后一个指针都应指向它。
- 如果链表中没有元素，第一个和最后一个指针应为 `None`。
- 如果对空队列进行出队操作，应返回 `None`。
- 我们可以假设这在内存中是可行的。

## 示例用法

### 入队

- 向空队列入队：如果队列为空，入队方法应将该元素添加为队列的第一个也是最后一个元素。
- 向非空队列入队：如果队列不为空，入队方法应将该元素添加到队列的末尾。

### 出队

- 对空队列进行出队操作 -> 返回 `None`：如果队列为空，出队方法应返回 `None`。
- 对只有一个元素的队列进行出队操作：如果队列只有一个元素，出队方法应移除该元素，并将第一个和最后一个指针设置为 `None`。
- 对有多个元素的队列进行出队操作：如果队列有多个元素，出队方法应移除第一个元素，并将第一个指针设置为指向下一个元素。
