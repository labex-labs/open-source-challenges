# 栈集合

## 问题

实现一个 `SetOfStacks` 类，它包装了一个栈列表，其中每个栈都有一个容量限制。该类应具备以下功能：

- 将一个元素压入列表中最后一个栈的顶部。如果最后一个栈已满，则创建一个新栈并将元素添加到新栈中。
- 从列表中最后一个栈弹出顶部元素。如果最后一个栈为空，则将其从列表中移除，并从列表中新的最后一个栈弹出顶部元素。如果列表为空，则返回 `None`。
- 从列表中特定的栈弹出一个元素。如果该栈为空，则将其从列表中移除。如果列表为空，则返回 `None`。

## 要求

`SetOfStacks` 类应满足以下要求：

- 该类应使用现有的栈类。
- 列表中的所有栈都应具有相同的容量限制。
- 如果一个栈已满，应自动创建一个新栈来存储额外的元素。
- 如果一个栈为空，应将其从列表中移除。
- 如果我们在一个空栈上执行弹出操作，该方法应返回 `None`。
- 实现应适合内存使用。

## 示例用法

`SetOfStacks` 类可用于以下场景：

- 在空栈上进行压入和弹出操作。
- 在非空栈上进行压入和弹出操作。
- 在达到容量限制的栈上进行压入操作以创建新栈。
- 在栈上进行弹出操作以销毁它。
