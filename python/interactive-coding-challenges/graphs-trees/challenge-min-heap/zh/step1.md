# 最小堆

## 问题

实现一个具有以下方法的最小堆：

- `extract_min()`：移除并返回堆中的最小值
- `insert(value)`：在保持堆属性的同时，将一个新值插入堆中

## 要求

实现应满足以下要求：

- 输入为整数
- 实现应能在内存中运行

## 示例用法

考虑以下最小堆：

```txt
          _5_
        /     \
       20     15
      / \    /  \
     22  40 25
```

- `extract_min()`：移除并返回堆中的最小值，即 5。得到的堆如下：

```txt
          _15_
        /      \
       20      25
      / \     /  \
     22  40
```

- `insert(2)`：在保持堆属性的同时，将值 2 插入堆中。得到的堆如下：

```txt
          _2_
        /     \
       20      5
      / \     / \
     22  40  25  15
```
