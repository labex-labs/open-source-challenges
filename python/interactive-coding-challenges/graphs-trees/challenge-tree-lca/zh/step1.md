# 树的最近公共祖先

## 问题

给定一棵二叉树和两个节点，找到它们的最近公共祖先。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 给定的树是一棵二叉树，而不是二叉搜索树。
- 我们不能假设这两个节点已经在树中。
- 我们可以假设二叉树能装入内存。

## 示例用法

考虑以下二叉树：

```txt
          _10_
        /      \
       5        9
      / \      / \
     12  3    18  20
        / \       /
       1   8     40
```

我们可以用以下输入和预期输出来测试我们的函数：

- 0, 5 -> 无（两个节点都不在树中）
- 5, 0 -> 无（两个节点都不在树中）
- 1, 8 -> 3
- 12, 8 -> 5
- 12, 40 -> 10
- 9, 20 -> 9
- 3, 5 -> 5
