# 反转二叉树

## 问题

给定一棵二叉树，编写一个函数来反转这棵树。该函数应以树的根节点作为输入，并返回反转后树的新根节点。

## 要求

要解决此问题，你需要满足以下要求：

- 你应该有一个表示二叉树中节点的 `Node` 类。
- 你应该交换二叉树中所有的左右节点对。
- 你应该优雅地处理无效输入。
- 你的解决方案应能在内存中运行。

## 示例用法

假设我们有以下二叉树：

```txt
     5
   /   \
  2     7
 / \   / \
1   3 6   9
```

反转树之后，我们应该得到：

```txt
     5
   /   \
  7     2
 / \   / \
9   6 3   1
```
