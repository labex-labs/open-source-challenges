# 树的高度

## 问题

给定一棵二叉树，编写一个 Python 函数来确定该树的高度。二叉树的高度是从根节点到树中任何叶节点的最长路径的长度。

## 要求

为了解决这个问题，我们需要满足以下要求：

- 给定的树是一棵二叉树。
- 我们已经有一个带有插入方法的节点类（Node class）。
- 解决方案要符合内存要求。

## 示例用法

以下是该函数应有的行为示例：

- 如果树只有一个节点，高度为 1。例如，如果输入是 5 -> 1，输出应为 1。
- 如果树有多个节点，高度是从根节点到任何叶节点的最长路径的长度。例如，如果输入是 5, 2, 8, 1, 3 -> 3，输出应为 3。
