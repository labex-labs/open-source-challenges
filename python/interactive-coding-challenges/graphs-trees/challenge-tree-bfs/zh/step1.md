# 树的广度优先搜索

## 问题

给定一棵二叉树，实现一个函数对该树进行广度优先遍历。在处理每个节点时，该函数应调用输入方法 `visit_func`。

## 要求

要解决此问题，必须满足以下要求：

- 已经有一个带有插入方法的 `Node` 类。
- 解决方案应能在内存中运行。
- 在处理每个节点时，应调用 `visit_func` 方法。

## 示例

### 广度优先遍历

假设我们有一棵具有以下结构的二叉树：

```
    5
   / \
  2   8
 / \
1   3
```

对该树进行广度优先遍历将导致按以下顺序访问节点：`5, 2, 8, 1, 3`。
