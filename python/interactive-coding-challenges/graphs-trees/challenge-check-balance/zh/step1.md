# 检查平衡性

## 问题

给定一棵二叉树，编写一个 Python 函数来判断它是否平衡。如果任何节点的两个子树的高度差最多为 1，则该二叉树被认为是平衡的。该函数应以二叉树的根节点作为输入，如果树是平衡的则返回 True，否则返回 False。如果输入为 None，则该函数应引发异常。

## 要求

为了解决这个问题，我们需要满足以下要求：

- 平衡树是指任何节点的两个子树的高度差不超过 1 的树。
- 如果输入为 None，函数应引发异常。
- 我们可以假设已经有一个带有插入方法的 Node 类。
- 我们可以假设程序在内存中运行正常。

## 示例用法

以下是该函数应有的一些行为示例：

- None -> 引发异常
- 1 -> True
- 5, 3, 8, 1, 4 -> True
- 5, 3, 8, 9, 10 -> False
