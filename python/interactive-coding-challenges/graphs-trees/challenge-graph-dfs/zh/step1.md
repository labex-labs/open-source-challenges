# 图的深度优先搜索

## 问题

在有向图上实现深度优先搜索。该算法应从给定节点开始，访问图中所有可达节点。访问节点的顺序应记录下来，并作为列表返回。

## 要求

要在有向图上实现深度优先搜索，必须满足以下要求：

- 图是有向的。
- 图和节点类已经实现。
- 图是连通的。
- 输入是有效的。
- 算法符合内存要求。

## 示例用法

假设我们有一个有向图，其边如下：

```
graph.add_edge(0, 1, 5)
graph.add_edge(0, 4, 3)
graph.add_edge(0, 5, 2)
graph.add_edge(1, 3, 5)
graph.add_edge(1, 4, 4)
graph.add_edge(2, 1, 6)
graph.add_edge(3, 2, 7)
graph.add_edge(3, 4, 8)
```

如果我们从节点 0 开始进行深度优先搜索，访问节点的顺序应该是 [0, 1, 3, 2, 4, 5]。
