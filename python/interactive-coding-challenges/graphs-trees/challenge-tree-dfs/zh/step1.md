# 树的深度优先搜索

## 问题

在二叉树上实现深度优先遍历（中序、前序、后序）。对于中序遍历，我们先访问左子树，然后是当前节点，最后是右子树。对于前序遍历，我们先访问当前节点，然后是左子树，最后是右子树。对于后序遍历，我们先访问左子树，然后是右子树，最后是当前节点。

## 要求

为了完成这个挑战，我们需要满足以下要求：

- 我们可以假设已经有一个带有插入方法的 `Node` 类。
- 当我们处理每个节点时，应该在该节点上调用输入方法 `visit_func`。
- 我们可以假设这能装入内存。

## 示例用法

以下是一些如何使用深度优先搜索算法的示例：

### 中序遍历

对于值为 5、2、8、1 和 3 的二叉树，中序遍历结果为 1、2、3、5 和 8。对于值为 1、2、3、4 和 5 的二叉树，中序遍历结果为 1、2、3、4 和 5。

### 前序遍历

对于值为 5、2、8、1 和 3 的二叉树，前序遍历结果为 5、2、1、3 和 8。对于值为 1、2、3、4 和 5 的二叉树，前序遍历结果为 1、2、3、4 和 5。

### 后序遍历

对于值为 5、2、8、1 和 3 的二叉树，后序遍历结果为 1、3、2、8 和 5。对于值为 1、2、3、4 和 5 的二叉树，后序遍历结果为 5、4、3、2 和 1。
