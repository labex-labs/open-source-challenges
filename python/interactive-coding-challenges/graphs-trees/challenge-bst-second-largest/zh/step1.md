# 二叉搜索树中的第二大节点

## 问题

给定一棵二叉搜索树，找到树中的第二大节点。如果输入为 `None` 或只有一个节点，则应引发异常。

为了解决这个问题，我们可以按特定顺序遍历树，并记录到目前为止我们看到的第二大节点。我们可以从遍历根节点的右子树开始，如果右子树为 `None`，那么最大节点就是根节点本身。如果右子树不为 `None`，我们可以继续遍历右子树，直到到达一个没有右子节点的节点。此时，树中的最大节点就是这个节点的父节点。如果这个父节点有左子节点，那么第二大节点就是父节点左子树中的最大节点。如果父节点没有左子节点，那么第二大节点就是父节点本身。

## 要求

这个挑战的要求如下：

- 如果输入为 `None` 或只有一个节点，则应引发异常。
  - 输入为 `None` 时应引发 `TypeError`。
  - 输入为单个节点时应引发 `ValueError`。
- 我们可以假设已经有一个带有插入方法的 `Node` 类。
- 我们可以假设这个问题在内存范围内。

## 示例用法

以下是一些如何使用此函数的示例：

- `None` 或单个节点 -> 异常

```txt
输入：
                _10_
              _/    \_
             5        15
            / \       / \
           3   8     12  20
          /     \         \
         2       4        30

输出：20

输入：
                 10
                 /
                5
               / \
              3   7
输出：7
```
