# 树的层级列表

## 问题

给定一棵二叉搜索树，为树的每一层创建一个列表。每个列表应包含该层的节点。这些列表应以数组的形式返回，其中每个子数组代表树的一层。

## 要求

要解决此问题，必须满足以下要求：

- 给定的树是一棵二叉搜索树。
- 树的每一层应由节点列表表示。
- 已经提供了一个带有插入方法的 `Node` 类。
- 解决方案应能在内存中运行。

## 示例用法

例如，给定具有以下值的二叉搜索树：

```
5, 3, 8, 2, 4, 1, 7, 6, 9, 10, 11
```

该函数应返回以下数组：

```
[[5], [3, 8], [2, 4, 7, 9], [1, 6, 10], [11]]
```

请注意，结果中的每个数字实际上是一个包含该数字的节点。
