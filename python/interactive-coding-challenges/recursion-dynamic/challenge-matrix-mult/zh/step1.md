# 矩阵乘法

## 问题

给定一系列 2x2 矩阵，我们需要找出将它们相乘的最小成本。两个矩阵相乘的成本是所需的标量乘法次数。例如，如果我们有矩阵 A、B 和 C，并且想要计算乘积 ABC，成本将是计算结果矩阵每个元素所需的标量乘法次数。

为了解决这个问题，我们需要找到矩阵相乘的最优顺序。我们相乘矩阵的顺序会影响乘法的总成本。例如，如果我们有矩阵 A、B 和 C，并且想要计算乘积 ABC，我们可以计算 (AB)C 或者 A(BC)。这两种计算的成本可能不同，我们需要找到使成本最小化的最优顺序。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 我们只需要计算矩阵乘法的成本，而不需要列出实际的操作顺序。
- 我们不能假设输入是有效的，需要处理无效输入。
- 我们可以假设问题适合内存。
