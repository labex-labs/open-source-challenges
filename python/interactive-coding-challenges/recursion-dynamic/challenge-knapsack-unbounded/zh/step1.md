# 无界背包

## 问题

给定一个具有总重量容量的背包，以及一个包含重量 w(i) 和价值 v(i) 的物品列表，确定你能携带的最大总价值。物品可以被多次选择。

## 要求

要解决无界背包问题，必须满足以下要求：

- 物品放入背包后可以被替换。
- 物品不能被拆分。
- 输入的物品重量和价值不能为 0。
- 只需要返回总价值，而不是构成最大总价值的物品。
- 输入可能无效，因此需要进行验证。
- 输入按价值/重量排序。
- 内存限制不是问题。

## 示例用法

无界背包问题可用于各种场景，如资源分配和金融投资组合优化。以下是一些它的使用示例：

- 如果总重量或物品为 None，应引发异常。
- 如果总重量或物品为 0，结果应为 0。
- 对于一般情况，假设总重量为 8，物品如下：

  | v   | w   |
  | --- | --- |
  | 0   | 0   |
  | 1   | 1   |
  | 3   | 2   |
  | 7   | 4   |

  能携带的最大价值为 14。
