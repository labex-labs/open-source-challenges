# 网格路径

## 问题

给定一个包含有效单元格和无效单元格的矩形网格，实现一个函数，为机器人找到一条从左上角移动到右下角的有效路径。如果没有有效路径，则返回 `None`。如果输入无效或矩阵为空，则返回 `None`。

## 要求

该算法的要求如下：

- 机器人只能向右和向下移动。
- 某些单元格可能是禁区。
- 网格是矩形的，没有参差不齐的情况。
- 机器人并不总是有有效的路径到达右下角。
- 输入并不总是有效的。
- 该算法应符合内存限制。

## 示例用法

考虑以下网格：

```txt
o = 有效单元格
x = 无效单元格

   0  1  2  3
0  o  o  o  o
1  o  x  o  o
2  o  o  x  o
3  x  o  o  o
4  o  o  x  o
5  o  o  o  x
6  o  x  o  x
7  o  x  o  o
```

- 一般情况：

```txt
预期结果 = [(0, 0), (1, 0), (2, 0),
            (2, 1), (3, 1), (4, 1),
            (5, 1), (5, 2), (6, 2),
            (7, 2), (7, 3)]
```

- 没有有效路径：在上述示例中，第 7 行第 2 列也是无效的 -> `None`
- 无输入 -> `None`
- 空矩阵 -> `None`
