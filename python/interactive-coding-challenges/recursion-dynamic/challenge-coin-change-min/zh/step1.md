# 找零最小化

## 问题

给定一组面值小于 n 美分的硬币，我们需要确定使用这些硬币凑成 n 美分的最少方法数。硬币可以以任何组合使用，并且每种面值的硬币数量是无限的。我们不需要报告代表最小值的硬币组合。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 硬币必须恰好凑成 n 美分。
- 我们可以假设我们有无限数量的硬币来凑成 n 美分。
- 我们不需要报告代表最小值的硬币组合。
- 我们不能假设硬币面值是按排序顺序给出的。
- 我们可以假设这适合内存。

## 示例用法

以下是这个函数的一些使用示例：

- 硬币：无 或 n：无 -> 异常
- 硬币：[] 或 n：0 -> 0
- 硬币：[1, 2, 3] 或 [3, 2, 1] -> 2
