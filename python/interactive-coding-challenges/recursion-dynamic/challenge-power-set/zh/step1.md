# 幂集

## 问题

给定一个集合，返回该集合的所有可能子集。子集应该是唯一的，这意味着如果两个子集具有相同的元素，它们应被视为同一个子集。空集也应作为一个子集包含在内。输入不一定是唯一的，并且我们不能假设输入是有效的。但是，我们可以假设该问题可以装入内存。

## 要求

要生成一个集合的幂集，我们需要满足以下要求：

- 生成的子集应该是唯一的，将具有相同元素的子集视为同一个。
- 空集应作为一个子集包含在内。
- 输入不一定是唯一的。
- 我们不能假设输入是有效的。
- 我们可以假设该问题可以装入内存。

## 示例用法

```txt
* 无 -> 无
* [] -> [[]]
* ['a'] -> [[],
            ['a']]
* ['a', 'b'] -> [[],
                 ['a'],
                 ['b'],
                 ['a', 'b']]
* ['a', 'b', 'c'] -> [[],
                      ['a'],
                      ['b'],
                      ['c'],
                      ['a', 'b'],
                      ['a', 'c'],
                      ['b', 'c'],
                      ['a', 'b', 'c']]
```
