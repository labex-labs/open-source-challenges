# 最长子串

## 问题

给定两个字符串，任务是找出最长公共子串。子串是连续的字符块。解决方案应区分大小写，并假设字符串为 ASCII 编码。输出应为字符串，并且程序应假设输入不一定有效。不过，它可以假设问题能在内存中解决。

## 要求

为了解决这个问题，程序必须满足以下要求：

- 输入不一定有效。
- 字符串为 ASCII 编码。
- 解决方案应区分大小写。
- 子串是连续的字符块。
- 输出应为字符串。
- 程序应假设问题能在内存中解决。

## 示例用法

程序应按以下方式运行：

- 如果 str0 或 str1 为 None，应引发异常。
- 如果 str0 或 str1 等于 0，输出应为空字符串。
- 一般情况下，给定 str0 = 'ABCDEFGHIJ' 和 str1 = 'FOOBCDBCDE'，输出应为 'BCDE'。
