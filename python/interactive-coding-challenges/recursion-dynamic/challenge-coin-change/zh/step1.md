# 硬币找零

## 问题

给定一组不同面额的硬币和总金额 `n`，确定用这些硬币凑成 `n` 美分的独特方式的总数。所提供的硬币面额均小于 `n` 美分。

## 要求

要解决此问题，必须满足以下要求：

- 硬币组合必须恰好凑成 `n` 美分。
- 可以假设使用无限数量的硬币来凑成 `n` 美分。
- 不需要报告表示最小组合的硬币组合。
- 硬币面额不是按排序顺序给出的。
- 解决方案必须能装入内存。

## 示例用法

以下示例展示了硬币找零问题的用法：

- 硬币：无 或 金额：无 -> 异常
- 硬币：[] 或 金额：0 -> 0
- 硬币：[1, 2, 3]，金额：5 -> 5
