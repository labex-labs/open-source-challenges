# Python 挑战：上楼梯问题

## 问题

假设你站在一个有 n 级台阶的楼梯底部。你每次可以走一步、两步或三步。问题是要找出走到第 n 级台阶有多少种可能的方式。

例如，如果有 3 级台阶，你可以按以下方式走上楼梯：

- 一步一步走（1 - 1 - 1）
- 先走一步再走两步（1 - 2）
- 先走两步再走一步（2 - 1）
- 直接走三步（3）

所以，走到第 3 级台阶有 4 种可能的方式。

## 要求

为了解决这个问题，我们需要记住以下要求：

- 如果 n == 0，结果应为 1。不过，对于这个问题有不同的处理方法，我们可以讨论一下。
- 我们不能假定输入是有效的。
- 我们可以假定这个问题在内存中是可以处理的。

## 示例用法

以下是一些使用 Python 解决这个问题的示例：

- 无输入或负输入 -> 引发异常
- n == 0 -> 1
- n == 1 -> 1
- n == 2 -> 2
- n == 3 -> 4
- n == 4 -> 7
- n == 10 -> 274
