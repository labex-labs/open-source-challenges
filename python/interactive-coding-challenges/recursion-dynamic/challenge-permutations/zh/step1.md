# 排列

## 问题

给定一个输入字符串，任务是找出该字符串中字符的所有可能排列。输出应为字符串列表，其中每个字符串代表输入字符串的一种唯一排列。输入字符串可能包含重复字符，但输出不应有任何重复。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 输入可能包含重复字符。
- 输出不应有任何重复。
- 输出应为字符串列表。
- 结果不需要排序。
- 输入并不总是有效的。
- 解决方案应能在内存中运行。

## 示例用法

以下是该函数应有的行为示例：

- 如果输入为 None，输出应为 None。
- 如果输入为空字符串，输出应为空字符串。
- 如果输入为 'AABC'，输出应为 ['AABC', 'AACB', 'ABAC', 'ABCA', 'ACAB', 'ACBA', 'BAAC', 'BACA', 'BCAA', 'CAAB', 'CABA', 'CBAA']。
