# 汉诺塔

## 问题

你的任务是用三根杆和 N 个圆盘实现汉诺塔。目标是将所有圆盘从第一根杆移动到第三根杆，同时遵循以下简单规则：

1. 一次只能移动一个圆盘。
2. 每次移动包括从一堆圆盘的顶部取出一个圆盘，并将其放置在另一堆圆盘的顶部或空杆上。
3. 任何圆盘都不能放在比它小的圆盘上面。

## 要求

要解决这个问题，你需要满足以下要求：

- 你应该有一个可用于此问题的栈类。
- 你应该在处理输入之前验证输入。
- 你应该确保程序在内存中运行正常。

## 示例用法

以下是程序应有的一些行为示例：

- 如果没有杆，应引发异常。
- 如果有 0 个圆盘，程序应返回 None。
- 如果只有 1 个圆盘，程序应将其从第一根杆移动到第三根杆。
- 如果有 2 个或更多圆盘，程序应将它们从第一根杆移动到第三根杆，同时遵循上述规则。
