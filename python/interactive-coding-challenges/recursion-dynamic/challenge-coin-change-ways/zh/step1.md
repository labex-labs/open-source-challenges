# 硬币找零方式

## 问题

给定一个整数 `n` 和一个由不同硬币组成的数组，编写一个函数来计算使用数组中的硬币凑出金额 `n` 的方式数量。一枚硬币可以使用任意次数，并且我们计算的是独特的组合。

例如，如果 `n = 4` 且硬币数组为 `[1, 2]`，则有 3 种找零方式：1 + 1 + 1 + 1、1 + 2 + 1 和 2 + 2。

## 要求

要解决这个问题，你需要：

- 编写一个函数，该函数接受两个参数：一个整数 `n` 和一个由不同硬币组成的数组。
- 使用动态规划来计算使用数组中的硬币凑出金额 `n` 的方式数量。
- 返回独特组合的数量。

## 示例

输入：`n = 4`，硬币数组 `coins = [1, 2]`

输出：3。1 + 1 + 1 + 1、1 + 2 + 1 和 2 + 2 是找零的方式。
