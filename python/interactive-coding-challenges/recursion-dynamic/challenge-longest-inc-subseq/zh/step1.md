# 最长递增子序列

## 问题

给定一个整数序列，找出最长递增子序列。该子序列可以是非连续的，并且可能包含重复项。如果有多个解决方案，返回其中任何一个。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 可能存在重复项吗？
  - 是
- 我们可以假设输入是整数吗？
  - 是
- 我们可以假设输入是有效的吗？
  - 不，我们需要处理无效输入。
- 我们期望结果是最长递增子序列的数组吗？
  - 是
- 我们可以假设这适合内存吗？
  - 是

## 示例用法

以下是这个函数的一些使用示例：

- 无 -> 异常
- [] -> []
- [3, 4, -1, 0, 6, 2, 3] -> [-1, 0, 2, 3]
