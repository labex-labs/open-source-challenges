# n 对括号

## 问题

问题是要找出 n 对括号的所有有效组合。有效组合是指每个左括号都有对应的右括号，并且括号对正确嵌套。例如，以下是 3 对括号的有效组合：

- ((()))
- (()())
- (())()
- ()(())
- ()()()

以下不是 3 对括号的有效组合：

- )()(
- ((()
- ))((
- ()()()

## 要求

为了解决这个问题，我们需要回答以下问题：

- 输入是表示括号对数的整数吗？
  - 是的，输入是表示括号对数的整数。
- 我们可以假设输入是有效的吗？
  - 不，我们不能假设输入是有效的。
- 输出是有效组合的列表吗？
  - 是的，输出是有效组合的列表。
- 输出应该有重复项吗？
  - 不，输出不应该有重复项。
- 我们可以假设这适合内存吗？
  - 是的，我们可以假设这适合内存。

## 示例用法

以下是该函数的示例用法：

- 无 -> 异常
- 负数 -> 异常
- 0 -> []
- 1 -> ['()']
- 2 -> ['(())', '()()']
- 3 -> ['((()))', '(()())', '(())()', '()(())', '()()()']
