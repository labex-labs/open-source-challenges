# 哈希映射

## 问题

实现一个具有插入、获取和删除方法的哈希表。哈希表应使用链地址法来解决冲突。键仅为整数。我们无需担心负载因子或验证输入。可以假设哈希表适合内存。

## 要求

- 键仅为整数。
- 使用链地址法解决冲突。
- 无需考虑负载因子。
- 无需验证输入。
- 哈希表适合内存。

## 示例用法

- `get` 方法：
  - 如果没有匹配的键，则引发 `KeyError` 异常。
  - 如果有匹配的键，则返回相应的值。
- `set` 方法：
  - 如果没有匹配的键，则将新的键值对添加到哈希表中。
  - 如果有匹配的键，则更新相应的值。
- `remove` 方法：
  - 如果没有匹配的键，则引发 `KeyError` 异常。
  - 如果有匹配的键，则从哈希表中删除相应的键值对。
