# 字符串差异

## 问题

给定两个字符串，我们需要找出它们之间唯一不同的字符。假设这两个字符串是 ASCII 编码且均为小写字母。我们不能假定输入是有效的，所以需要检查是否为 None。如果输入有效，我们可以假定两个字符串之间只有一个不同的字符。我们还需要确保解决方案在内存中可行。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 字符串是 ASCII 编码。
- 字符串是小写字母。
- 我们需要检查输入是否为 None。
- 两个字符串之间只有一个不同的字符。
- 解决方案应在内存中可行。

## 示例用法

以下是一些如何使用该函数的示例：

- 输入为 None -> 类型错误
- 'ab', 'aab' -> 'a'
- 'aab', 'ab' -> 'a'
- 'abcd', 'abcde' -> 'e'
- 'aaabbcdd', 'abdbacade' -> 'e'
