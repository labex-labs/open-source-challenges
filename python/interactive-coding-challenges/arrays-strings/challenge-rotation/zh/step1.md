# 旋转

## 问题

给定两个字符串 s1 和 s2，通过调用（仅一次）一个函数 is_substring 来确定 s1 是否是 s2 的旋转。函数 is_substring 接受两个字符串作为输入，如果第一个字符串是第二个字符串的子串，则返回 True，否则返回 False。

## 要求

为了解决这个问题，我们需要满足以下要求：

- 字符串为 ASCII 编码。
- 比较区分大小写。
- 我们可以使用额外的数据结构。
- 字符串能够存入内存。

## 示例用法

以下是该函数应有的一些行为示例：

- 如果两个字符串长度不同，函数应返回 False。
- 如果任何一个字符串为 None，函数应返回 False。
- 如果其中一个字符串是空格而另一个不是，函数应返回 False。
- 如果两个字符串都是空格，函数应返回 True。
- 如果 s1 是 s2 的旋转，函数应返回 True。
