# 压缩

## 问题

给定一个字符串，对其进行压缩，使得 'AAABCCDDDD' 变为 'A3BC2D4'。只有当压缩后的字符串节省空间时，才返回它。如果压缩后的字符串比原始字符串长，则返回原始字符串。该字符串区分大小写，并且可以假定为 ASCII 编码。可以使用额外的数据结构，并且可以假定字符串能装入内存。

## 要求

必须满足以下要求：

- 字符串为 ASCII 编码。
- 字符串区分大小写。
- 可以使用额外的数据结构。
- 字符串能装入内存。

## 示例用法

以下示例展示了该函数预期的输入和输出：

- 无 -> 无
- '' -> ''
- 'AABBCC' -> 'AABBCC'
- 'AAABCCDDDD' -> 'A3BC2D4'
