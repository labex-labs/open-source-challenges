# 反转字符串

## 问题

实现一个函数，原地反转一个字符串（字符列表）。这意味着该函数应修改原始字符串，而不是创建一个新的字符串。该函数应以字符列表作为输入，并返回字符顺序反转后的同一个列表。

为了解决这个问题，我们需要考虑几个要求：

## 要求

- 可以假设字符串为 ASCII 编码。
- 由于我们需要原地完成此操作，因此不能使用切片运算符或 reversed 函数。
- 由于 Python 字符串是不可变的，我们可以使用字符列表来代替。

## 示例用法

以下是该函数应有的行为示例：

- 无 -> 无
- [''] -> ['']
- ['f', 'o', 'o','', 'b', 'a', 'r'] -> ['r', 'a', 'b','', 'o', 'o', 'f']
