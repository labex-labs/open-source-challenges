# 找到循环起始点

## 问题

给定一个单链表，如果存在循环，我们需要找到循环的起始点。循环定义为链表中的一个节点指向之前的一个节点，从而形成一个环。如果没有循环，我们返回 `None`。如果存在循环，我们返回循环开始的节点。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 链表是单链表。
- 我们不能假设总是会传入一个循环链表。
- 当我们找到一个循环时，我们返回循环开始的节点。
- 我们可以假设已经有一个可用于此问题的链表类。

## 示例用法

以下是这个函数的一些使用示例：

- 空链表 -> `None`
- 非循环链表 -> `None`
  - 一个元素
  - 两个或更多元素
- 循环链表一般情况
