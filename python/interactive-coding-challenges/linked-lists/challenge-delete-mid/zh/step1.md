# 删除中间节点

## 问题

给定一个非循环的单链表，在仅能访问要删除节点的情况下，删除链表中间的节点。如果要删除的是最后一个节点，则将其设为值为 None 的哑节点。你可以假设我们已经有一个可用于此问题的链表类。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 链表是非循环的单链表。
- 如果要删除的是最后一个节点，则将其设为值为 None 的哑节点。
- 我们已经有一个可用于此问题的链表类。

## 示例用法

以下是一些如何使用此函数的示例：

- 在空链表上删除应返回 None。
- 删除 None 应返回 None。
- 在只有一个节点的链表上删除应返回 [None]。
- 在有多个节点的链表上删除应返回删除节点后的更新链表。
