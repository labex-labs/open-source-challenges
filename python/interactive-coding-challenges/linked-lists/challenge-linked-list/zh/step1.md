# 链表

## 问题

实现一个具有以下方法的链表：

- insert(value)：在链表头部插入一个具有给定值的新节点
- append(value)：在链表尾部插入一个具有给定值的新节点
- find(value)：返回链表中第一个具有给定值的节点，如果不存在这样的节点，则返回 None
- delete(value)：删除链表中第一个具有给定值的节点，如果不存在这样的节点，则不做任何操作
- length()：返回链表中的节点数量
- print()：打印链表中所有节点的值，值之间用空格分隔

## 要求

链表实现应满足以下要求：

- 链表是非循环的单链表。
- 实现只跟踪链表的头节点，而不是尾节点。
- 不能将 None 值插入链表。

## 示例用法

### 插入到头部

- 插入 None：引发错误，因为不能将 None 值插入链表。
- 在空链表中插入：将值作为链表的第一个节点插入。
- 在有一个或多个元素的链表中插入：将值作为链表的第一个节点插入，将现有节点向右移动。

### 追加

- 追加 None：引发错误，因为不能将 None 值插入链表。
- 在空链表中追加：将值作为链表的第一个节点插入。
- 在有一个或多个元素的链表中追加：将值作为链表的最后一个节点插入，更新前一个最后一个节点的引用以指向新节点。

### 查找

- 查找 None：返回 None，因为在链表中找不到 None 值。
- 在空链表中查找：返回 None，因为链表中没有节点。
- 在有一个或多个匹配元素的链表中查找：返回链表中第一个具有给定值的节点。
- 在没有匹配项的链表中查找：返回 None，因为链表中没有具有给定值的节点。

### 删除

- 删除 None：不做任何操作，因为不能从链表中删除 None 值。
- 在空链表中删除：不做任何操作，因为链表中没有节点。
- 在有一个或多个匹配元素的链表中删除：删除链表中第一个具有给定值的节点，将现有节点向左移动。
- 在没有匹配项的链表中删除：不做任何操作，因为链表中没有具有给定值的节点。

### 长度

- 零个或多个元素的长度：返回链表中的节点数量。

### 打印

- 打印空链表：不打印任何内容。
- 打印有一个或多个元素的链表：打印链表中所有节点的值，值之间用空格分隔。
