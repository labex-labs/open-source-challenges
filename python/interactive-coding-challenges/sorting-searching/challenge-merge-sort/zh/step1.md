# 归并排序

## 问题

用 Python 实现归并排序。给定一个未排序的数组，该算法应将数组按升序排序。该算法应能够处理任意长度的数组，并且适用于整数和字符串。

## 要求

要在 Python 中实现归并排序，必须满足以下要求：

- 该算法应采用分治方法。
- 该算法应能够处理任意长度的数组。
- 该算法应适用于整数和字符串。
- 该算法应将数组按升序排序。
- 该算法的时间复杂度应为 O(n log n)。
- 该算法应以 Python 实现。

## 示例用法

以下示例演示了归并排序的用法：

- 无输入 -> 异常：如果输入为 None，该算法应引发异常。
- 空输入 -> []：如果输入是一个空数组，该算法应返回一个空数组。
- 一个元素 -> [元素]：如果输入仅包含一个元素，该算法应返回相同的数组。
- 两个或更多元素：该算法应将数组按升序排序。如果左子数组和右子数组的长度不同，该算法仍应能够正确地对数组进行排序。
