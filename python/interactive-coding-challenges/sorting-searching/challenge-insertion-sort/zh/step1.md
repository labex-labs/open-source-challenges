# 插入排序

## 问题

问题是要用 Python 实现插入排序。给定一个未排序的元素列表，该算法应将列表按升序排序。该算法的工作方式是遍历列表，并将每个元素插入到列表已排序部分的正确位置。

该算法首先假设列表中的第一个元素已经排序。然后，它遍历列表中的其余元素，将每个元素与列表已排序部分中的元素进行比较。如果该元素小于已排序部分中的当前元素，则将其插入到该元素之前。如果该元素大于已排序部分中的所有元素，则将其插入到已排序部分的末尾。

## 要求

要用 Python 实现插入排序，应满足以下要求：

- 一个简单的解决方案就足够了。
- 允许重复。
- 输入不一定有效，因此算法应能优雅地处理无效输入。
- 算法应能在内存中运行。

## 示例用法

以下是一些使用插入排序算法的示例：

- 无输入 -> 异常：如果输入为 None，则应引发异常。
- 空输入 -> []：如果输入是一个空列表，输出也应为空列表。
- 一个元素 -> [元素]：如果输入是一个只有一个元素的列表，输出应是同一个列表。
- 两个或更多元素：如果输入是一个有两个或更多元素的列表，输出应是一个按升序排序的列表。
