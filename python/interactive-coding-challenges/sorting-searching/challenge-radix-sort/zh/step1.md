# 基数排序

## 问题

实现基数排序算法，对整数列表进行排序。该算法通过从最低有效位到最高有效位比较整数的各个数位来进行排序。算法首先根据最低有效位对整数进行排序，然后是次低有效位，依此类推，直到最高有效位被排序。

## 要求

要实现基数排序，必须满足以下要求：

- 输入必须是整数列表。
- 检查是否存在 `None` 代替数组的情况。
- 假设数组元素为整数。
- 数位必须是十进制。
- 算法必须能够处理任意数量的数位。
- 算法必须能在内存中运行。

## 示例用法

以下是如何使用基数排序算法的示例：

- `None` -> 异常
- `[]` -> `[]`
- `[128, 256, 164, 8, 2, 148, 212, 242, 244]` -> `[2, 8, 128, 148, 164, 212, 242, 244, 256]`
