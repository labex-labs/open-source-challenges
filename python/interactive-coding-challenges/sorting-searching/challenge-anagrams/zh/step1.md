# 变位词

## 问题

给定一个字符串数组，编写一个函数对该数组进行排序，使得所有变位词都相邻。变位词定义为通过重新排列另一个单词或短语的字母而形成的单词或短语。例如，“act”和“cat”是彼此的变位词。

## 要求

要解决此问题，必须满足以下要求：

- 函数必须在排序后的数组中将所有变位词分组在一起。
- 除了变位词的分组外，没有其他排序要求。
- 输入可能无效，因此函数必须处理无效输入。
- 函数必须在内存中运行。

## 示例用法

在这些场景中，函数应按以下方式运行：

- 无 -> 异常
- [] -> []
- 一般情况
  - 输入：['ram', 'act', 'arm', 'bat', 'cat', 'tab']
  - 结果：['arm', 'ram', 'act', 'cat', 'bat', 'tab']
