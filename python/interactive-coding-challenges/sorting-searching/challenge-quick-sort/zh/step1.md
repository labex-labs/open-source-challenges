# 快速排序

## 问题

用 Python 实现快速排序算法。该算法应以一个未排序的列表作为输入，并返回一个已排序的列表。该算法应适用于任何大小的列表，包括空列表和包含重复元素的列表。该算法还应能优雅地处理无效输入。

快速排序算法的工作原理如下：

1. 从列表中选择一个枢轴元素。
2. 将列表划分为两个子列表：一个包含小于枢轴的元素，另一个包含大于枢轴的元素。
3. 对这些子列表递归地应用快速排序算法。
4. 将已排序的子列表与中间的枢轴元素连接起来。

## 要求

要在 Python 中实现快速排序算法，应满足以下要求：

- 该算法不应是原地排序解决方案。
- 该算法应能处理列表中的重复元素。
- 该算法应能处理无效输入，如 None 或非列表输入。
- 该算法应能处理任何大小的列表，包括空列表。

## 示例用法

以下是一些在 Python 中使用快速排序算法的示例：

- None -> 异常

```python
quick_sort(None)
```

- 空输入 -> []

```python
quick_sort([])
```

- 一个元素 -> [元素]

```python
quick_sort([5])
```

- 两个或更多元素

```python
quick_sort([5, 2, 8, 3, 1, 9])
```
