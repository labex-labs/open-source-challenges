# 旋转数组搜索

## 问题

给定一个已排序且被旋转了若干次的数组，我们需要在该数组中找到一个特定元素。例如，若原始的已排序数组为 [1, 2, 3, 4, 5]，且它被旋转了两次后变为 [3, 4, 5, 1, 2]，我们需要找到特定元素（如 4）的索引。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 输入是一个整数数组。
- 我们不知道数组被旋转了多少次。
- 数组最初是按升序排序的。
- 对于输出，我们需要返回正在搜索的元素的索引。
- 我们不能假设输入是有效的。
- 我们可以假设解决方案适合内存。

## 示例用法

以下是该函数使用方法的一些示例：

- 如果未提供输入，应引发异常。
- 如果提供的是一个空数组，应返回 None。
- 如果在数组中未找到我们正在搜索的元素，应返回 None。
- 如果数组包含重复项，该函数仍应能够找到正确的索引。
- 如果数组不包含重复项，该函数仍应能够找到正确的索引。
