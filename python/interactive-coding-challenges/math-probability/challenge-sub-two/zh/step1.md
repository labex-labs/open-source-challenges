# 相减两个数

## 问题

编写一个 Python 函数，该函数接受两个整数作为输入，并在不使用“+”或“-”符号的情况下返回它们的差。该函数应处理以下情况：

- 如果任一输入为 None，函数应引发 TypeError。
- 该函数应适用于正整数和负整数。

## 要求

为了解决这个问题，我们需要遵循以下要求：

- 检查输入是否为 None，必要时引发 TypeError。
- 我们可以假设输入将适合内存。

## 示例用法

以下是该函数应有的行为示例：

```
sub_two(None, 5) -> TypeError
sub_two(7, 5) -> 2
sub_two(-5, -7) -> 2
sub_two(-5, 7) -> -12
sub_two(5, -7) -> 12
```
