# 各位相加

## 问题

给定一个整数，我们需要重复将其各位数字相加，直到结果为个位数。例如，给定整数 138，我们计算 1 + 3 + 8 = 12。由于 12 不是个位数，我们重复此过程并计算 1 + 2 = 3。因此，最终结果是 3。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 输入的整数是非负的。
- 输入并不总是有效的，所以我们需要处理可能出现的任何错误。
- 解决方案应能在内存中运行。

## 示例用法

以下是该函数的一些使用示例：

- 如果我们输入 None，函数应引发 TypeError。
- 如果我们输入一个负整数，函数应引发 ValueError。
- 如果我们输入 9，函数应返回 9，因为它已经是个位数。
- 如果我们输入 138，函数应返回 3。
- 如果我们输入 65536，函数应返回 7。
