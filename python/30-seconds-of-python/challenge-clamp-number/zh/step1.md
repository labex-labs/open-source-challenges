# 钳位数字挑战

## 问题

编写一个函数 `clamp_number(num, a, b)`，它接受三个参数：

- `num`（整数或浮点数）：要钳位的数字
- `a`（整数或浮点数）：范围的下限
- `b`（整数或浮点数）：范围的上限

该函数应将 `num` 钳位在边界值指定的包含范围内。如果 `num` 落在范围 (`a`, `b`) 内，则返回 `num`。否则，返回该范围内最接近的数字。

## 示例

```python
clamp_number(2, 3, 5) # 3
clamp_number(1, -1, -5) # -1
clamp_number(10, 1, 5) # 5
clamp_number(-10, -5, -1) # -5
```
