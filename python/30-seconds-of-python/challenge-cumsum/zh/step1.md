# 部分和列表

## 问题

编写一个函数 `partial_sum(lst)`，它接受一个数字列表作为参数，并返回一个部分和的列表。你的函数应执行以下步骤：

1. 使用 `itertools.accumulate()` 为列表中的每个元素创建累积和。
2. 使用 `list()` 将结果转换为列表。
3. 返回部分和的列表。

## 示例

```python
partial_sum([1, 2, 3, 4, 5]) # [1, 3, 6, 10, 15]
partial_sum([2, 4, 6, 8, 10]) # [2, 6, 12, 20, 30]
partial_sum([5, 10, 15, 20, 25]) # [5, 15, 30, 50, 75]
```
