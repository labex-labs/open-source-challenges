# 展开列表

## 问题

编写一个名为 `spread(arg)` 的函数，该函数接受一个列表作为参数，并返回一个新列表，其中包含原始列表中所有扁平化后的元素。如果原始列表中的某个元素本身也是一个列表，则其元素应单独添加到新列表中。该函数不应修改原始列表。

为了实现这个函数，你应该遍历原始列表的元素，并使用展开运算符将元素添加到新列表中。如果某个元素是列表，你应该使用 `extend()` 方法将其元素添加到新列表中。如果某个元素不是列表，你应该使用 `append()` 方法将其添加到新列表中。

## 示例

```python
spread([1, 2, 3, [4, 5, 6], [7], 8, 9]) # [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
