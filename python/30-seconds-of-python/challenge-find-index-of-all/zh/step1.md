# 找到所有匹配的索引

## 问题

编写一个函数 `find_index_of_all(lst, fn)`，它接受一个列表 `lst` 和一个测试函数 `fn` 作为参数，并返回 `lst` 中所有使得 `fn` 返回 `True` 的元素的索引列表。

### 输入

- 一个整数列表 `lst`。
- 一个测试函数 `fn`，它接受一个整数作为输入并返回一个布尔值。

### 输出

- 一个整数列表，表示 `lst` 中所有使得 `fn` 返回 `True` 的元素的索引。

## 示例

```python
find_index_of_all([1, 2, 3, 4], lambda n: n % 2 == 1) # [0, 2]
find_index_of_all([1, 2, 3, 4], lambda n: n > 2) # [2, 3]
find_index_of_all([1, 2, 3, 4], lambda n: n < 0) # []
```

### 注意

- 在第一个示例中，测试函数 `lambda n: n % 2 == 1` 检查整数是否为奇数。该函数返回 `[0, 2]`，因为列表 `[1, 2, 3, 4]` 中索引为 `0` 和 `2` 的元素是奇数。
- 在第二个示例中，测试函数 `lambda n: n > 2` 检查整数是否大于 `2`。该函数返回 `[2, 3]`，因为列表 `[1, 2, 3, 4]` 中索引为 `2` 和 `3` 的元素大于 `2`。
- 在第三个示例中，测试函数 `lambda n: n < 0` 检查整数是否为负数。该函数返回 `[]`，因为列表 `[1, 2, 3, 4]` 中没有负数元素。
