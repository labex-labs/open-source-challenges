# 基于函数的列表交集

## 问题

编写一个函数 `intersection_by(a, b, fn)`，它接受两个列表 `a` 和 `b`，以及一个函数 `fn`。该函数应在将提供的函数应用于两个列表的每个元素后，返回两个列表中都存在的元素列表。

### 输入

- 两个列表 `a` 和 `b` (1 <= len(a), len(b) <= 1000)
- 一个函数 `fn`，它接受一个参数并返回一个值

### 输出

- 一个列表，其中包含在将提供的函数应用于两个列表的每个元素后，两个列表中都存在的元素。

## 示例

```python
intersection_by([2.1, 1.2], [2.3, 3.4], floor) # [2.1]
```

### 注意

在上面的示例中，函数 `floor()` 应用于两个列表中的每个元素。得到的集合分别是 `{2, 3}` 和 `{2, 1}`。这些集合的交集是 `{2}`，然后作为列表返回。
