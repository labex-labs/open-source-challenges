# 填充数字

## 问题

编写一个函数 `pad_number(n, l)`，它接受一个数字 `n` 和一个长度 `l`，并返回一个表示填充后数字的字符串。该函数应该用前导零填充数字，使其长度为 `l` 位。如果数字本身已经是 `l` 位长，则函数应将该数字作为字符串返回。

为了填充数字，你可以使用 `str.zfill()` 方法。该方法接受一个长度，并在字符串前填充零，直到达到该长度。例如，`"7".zfill(6)` 将返回 `"000007"`。

## 示例

```python
pad_number(1234, 6) # '001234'
pad_number(7, 6) # '000007'
pad_number(123456789, 9) # '123456789'
```
