# 测试列表中是否有元素为真值

## 问题

编写一个函数 `some(lst, fn)`，它接受一个列表 `lst` 和一个函数 `fn` 作为参数。如果函数 `fn` 对列表 `lst` 中的至少一个元素返回 `True`，则该函数应返回 `True`。如果列表中没有元素满足该条件，则函数应返回 `False`。如果未提供函数，则该函数应使用恒等函数（即返回元素本身的函数）。

## 示例

```python
some([0, 1, 2, 0], lambda x: x >= 2 ) # 为真
some([0, 0, 1, 0]) # 为真
some(['', 'hello', 'world'], bool) # 为真
some(['', '', ''], bool) # 为假
```
