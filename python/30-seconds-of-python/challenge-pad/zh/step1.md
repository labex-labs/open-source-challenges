# 填充字符串

## 问题

编写一个函数 `pad(s: str, length: int, char: str ='') -> str`，如果字符串比指定长度短，则在其两边用指定字符填充。该函数应接受三个参数：

- `s`：需要填充的字符串
- `length`：指定填充后字符串的总长度的整数
- `char`：用于填充字符串的字符。默认值是一个空白字符

该函数应返回填充后的字符串。

## 示例

```python
pad('cat', 8) #' cat   '
pad('42', 6, '0') # '004200'
pad('foobar', 3) # 'foobar'
```
