# 根据函数找到列表中的最小值

## 问题

编写一个名为 `min_by(lst, fn)` 的函数，该函数接受一个列表 `lst` 和一个函数 `fn` 作为参数。该函数应使用提供的函数将列表中的每个元素映射到一个值，然后返回最小值。

## 示例

```python
min_by([{ 'n': 4 }, { 'n': 2 }, { 'n': 8 }, { 'n': 6 }], lambda v : v['n']) # 2
```

在上面的示例中，`min_by()` 函数被调用，参数为一个字典列表和一个 lambda 函数，该 lambda 函数从每个字典中提取 `'n'` 键的值。该函数返回列表中的最小值，即 `2`。
