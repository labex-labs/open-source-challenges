# 岭回归

岭回归是一种线性回归方法，它在普通最小二乘目标函数中添加了一个惩罚项。这个惩罚项通过将系数向零收缩来帮助减少过拟合。模型的复杂度可以通过正则化参数来控制。

让我们拟合一个岭回归模型。

```python
reg = linear_model.Ridge(alpha=0.5)
reg.fit([[0, 0], [0, 0], [1, 1]], [0, 0.1, 1])

print(reg.coef_)
```

- 我们创建一个 `Ridge` 实例，将正则化参数 `alpha` 设置为 0.5。
- 我们使用 `fit` 方法将模型拟合到训练数据上。
- 我们打印岭回归模型的系数。
