# 简介

超参数是估计器无法直接学习的参数。它们作为参数传递给估计器类的构造函数。调整估计器的超参数是构建有效机器学习模型的重要一步。这涉及找到能使模型性能最佳的超参数的最优组合。

Scikit-learn 提供了几种搜索最佳超参数的工具：`GridSearchCV` 和 `RandomizedSearchCV`。在本实验中，我们将逐步介绍使用这些工具调整超参数的过程。
