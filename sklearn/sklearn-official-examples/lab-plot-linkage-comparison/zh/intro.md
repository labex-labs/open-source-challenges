# 简介

在本实验中，我们将使用 Python 的 scikit-learn 库对一些简单的数据集执行层次聚类。层次聚类是一种聚类方法，你可以通过自上而下或自下而上的方式构建聚类层次结构。层次聚类的目标是找到彼此相似且与其他聚类中的点不同的点聚类。
