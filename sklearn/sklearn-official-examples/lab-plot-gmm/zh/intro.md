# 简介

本实验将指导你使用 Python 中的 scikit-learn 库实现高斯混合模型（Gaussian Mixture Models，GMMs）。GMMs 是概率模型，假设数据是由多个高斯分布的混合生成的。它们广泛应用于计算机视觉、金融和生物信息学等各个领域，用于聚类和密度估计任务。
