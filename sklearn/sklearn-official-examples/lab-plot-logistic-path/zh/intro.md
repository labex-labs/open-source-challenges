# 简介

L1 逻辑回归模型是一种二元分类方法，它使用 L1 正则化来促使模型产生稀疏性。该模型的正则化路径展示了随着正则化强度增加时模型的系数。在本实验中，我们将使用鸢尾花数据集来训练 L1 惩罚逻辑回归模型，并绘制它们的正则化路径。
