# 简介

本实验展示了如何使用 scikit-learn 库通过交叉验证进行超参数调优。目的是使用二元分类对手写数字图像进行分类，以便于理解：识别一个数字是否为 8。所使用的数据集是数字数据集。然后，在模型选择步骤中未使用的专用评估集上测量所选超参数和训练模型的性能。
