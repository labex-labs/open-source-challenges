# 简介

本实验展示了如何在 Python 的 Scikit-learn 库中使用不同的核函数进行高斯过程回归（GPR）。GPR 是一种非参数回归技术，它可以将复杂模型拟合到带有噪声的数据上。核函数用于确定任意两个输入点之间的相似度。核函数的选择很重要，因为它决定了拟合数据的模型形状。在本实验中，我们将介绍 GPR 中最常用的核函数。
