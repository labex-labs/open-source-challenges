# 简介

在本实验中，我们将使用梯度提升（Gradient Boosting）为糖尿病回归任务构建一个预测模型。我们将在糖尿病数据集上训练该模型，并使用最小二乘损失和500棵深度为4的回归树，从`sklearn.ensemble.GradientBoostingRegressor`中获得结果。
