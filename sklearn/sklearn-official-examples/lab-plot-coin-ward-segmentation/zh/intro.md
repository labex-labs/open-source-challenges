# 简介

在本实验中，我们将学习如何使用层次聚类对二维图像进行分割。层次聚类是一种将相似数据点分组在一起的聚类算法。在图像分割的背景下，层次聚类可用于将具有相似颜色强度的像素分组在一起，这在识别图像中不同的区域或对象时可能会很有用。

我们将使用 Python 的 scikit-learn 库对硬币图像执行层次聚类。
