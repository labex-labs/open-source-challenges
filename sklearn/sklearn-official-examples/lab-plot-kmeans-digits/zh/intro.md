# 简介

在本实验中，我们将使用 Python 中的 scikit-learn 库探索 K 均值聚类算法。我们将使用手写数字数据集，该数据集包含 64 个特征，代表一个数字的 8x8 图像，并且我们将尝试根据它们所代表的数字将图像分组在一起。我们将比较 K 均值的不同初始化方法，并使用各种指标评估聚类的性能。
