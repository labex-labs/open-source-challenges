# 简介

Scikit-learn 是一个广受欢迎的 Python 库，它为机器学习任务提供了简单高效的 API。Scikit-learn 的关键特性之一是其内置的可视化 API，这使得为机器学习模型生成可视化变得轻而易举。在本实验中，我们将探索如何使用 Scikit-learn 可视化 API 来比较两个不同分类器的 ROC 曲线。
