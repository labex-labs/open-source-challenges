# 简介

本实验展示了如何使用 Python 中的 Scikit-learn 库执行具有连接性约束的层次聚类。在层次聚类中，簇是通过根据簇之间的距离递归合并或分裂来形成的。连接性约束可用于根据数据点之间的连接性来限制簇的形成，这可以产生更有意义的簇。
