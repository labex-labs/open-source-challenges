# 简介

在本实验中，我们将学习如何使用 Python 的 scikit-learn 库连接多种特征提取方法。我们将使用 `FeatureUnion` 变换器来组合通过主成分分析（PCA）和单变量选择获得的特征。使用此变换器组合特征的好处是，它允许在整个过程中进行交叉验证和网格搜索。
