# 创建自定义核函数

在这一步中，我们将创建一个自定义核函数。该自定义核函数将是两个矩阵的点积。我们将创建一个值为 [[2, 0], [0, 1.0]] 的矩阵 M。然后，我们将矩阵 X 和 Y 与 M 相乘，并取它们的点积。

```python
def my_kernel(X, Y):
    """
    我们创建一个自定义核函数：

                 (2  0)
    k(X, Y) = X  (    ) Y.T
                 (0  1)
    """
    M = np.array([[2, 0], [0, 1.0]])
    return np.dot(np.dot(X, M), Y.T)
```
