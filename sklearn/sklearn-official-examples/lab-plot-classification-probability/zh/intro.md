# 简介

本实验演示了如何使用 Python 的 Scikit-learn 绘制不同分类器的分类概率。我们将使用一个三类数据集，并用支持向量分类器、具有一对多或多项式设置的 L1 和 L2 惩罚逻辑回归以及高斯过程分类对其进行分类。
