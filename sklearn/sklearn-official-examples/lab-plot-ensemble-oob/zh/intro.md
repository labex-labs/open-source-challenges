# 简介

在本实验中，我们将演示如何使用 Python 的 scikit-learn 库来测量随机森林模型的袋外（OOB）错误率。OOB 错误率是指使用各自自助抽样样本中不包含该观测值的树的预测结果计算出的每个训练观测值的平均误差。这使得随机森林模型在训练时就能进行拟合和验证。
