# 简介

在本实验中，我们将学习如何使用 Scikit-Learn 库的 `make_multilabel_classification` 函数生成多标签数据集。该函数生成多标签数据的随机样本，其中每个样本有两个特征的计数，这两个特征在两个类别中的每一个中分布不同。
