# 简介

在本实验中，我们将探索 `newtype` 习惯用法，它通过允许我们创建一个与其底层类型不同的新类型来提供编译时保证。这里给出了一个示例，其中结构体 `Years` 用于表示以年为单位的年龄，结构体 `Days` 用于表示以天为单位的年龄。通过使用 `newtype` 习惯用法，我们可以确保向程序提供正确类型的值，例如在年龄验证函数 `old_enough` 中，该函数需要一个 `Years` 类型的值。此外，我们还将学习如何使用元组或解构语法将 `newtype` 的值作为其底层类型获取。

> **注意**：如果实验未指定文件名，你可以使用任何你想要的文件名。例如，你可以使用 `main.rs`，并通过 `rustc main.rs &&./main` 进行编译和运行。
