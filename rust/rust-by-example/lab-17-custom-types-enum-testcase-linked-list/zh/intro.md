# 简介

在本实验中，我们使用 Rust 中的 `enum` 实现了一个链表。`List` 枚举有两个变体：`Cons`，它表示一个带有元素和指向下一个节点指针的节点；`Nil`，它表示链表的末尾。该枚举有诸如 `new` 方法来创建一个空列表，`prepend` 方法在列表前端添加一个元素，`len` 方法返回列表的长度，以及 `stringify` 方法返回列表的字符串表示形式。提供的主函数演示了如何使用这些方法来创建和操作链表。

> **注意**：如果实验未指定文件名，你可以使用任何你想要的文件名。例如，你可以使用 `main.rs`，并通过 `rustc main.rs &&./main` 进行编译和运行。
