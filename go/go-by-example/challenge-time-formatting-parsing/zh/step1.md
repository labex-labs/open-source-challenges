# 时间格式化与解析

问题在于使用提供的布局在 Go 语言中格式化和解析时间。

## 要求

- 使用 `time` 包来格式化和解析时间。
- 使用 `time.RFC3339` 布局来格式化和解析时间。
- 使用 `Mon Jan 2 15:04:05 MST 2006` 这个参考时间来展示格式化/解析给定时间/字符串的模式。
- 使用 `Parse` 函数来解析时间。
- 使用 `Format` 函数来格式化时间。
- 使用 `fmt.Println` 函数来打印格式化后的时间。
- 使用 `fmt.Printf` 函数来打印带有提取组件的格式化时间。

## 示例

```sh
$ go run time-formatting-parsing.go
2014-04-15T18:00:15-07:00
2012-11-01 22:08:41 +0000 +0000
6:00PM
Tue Apr 15 18:00:15 2014
2014-04-15T18:00:15.161182-07:00
0000-01-01 20:41:00 +0000 UTC
2014-04-15T18:00:15-00:00
解析时间 "8:41PM" 为 "Mon Jan _2 15:04:05 2006":...
```
