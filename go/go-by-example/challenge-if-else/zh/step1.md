# if-else

你需要完成 `checkNumber` 函数，该函数接受一个整数作为输入并返回一个字符串。如果数字是偶数，则返回 "even"，否则返回 "odd"。

## 要求

- 函数应命名为 `checkNumber`。
- 函数应接受一个整数作为输入。
- 函数应返回一个字符串。
- 如果数字是偶数，则返回 "even"。
- 如果数字是奇数，则返回 "odd"。

## 示例

```sh
$ go run if-else.go
7 是奇数
8 能被 4 整除
9 是个一位数

# Go 语言中没有 [三元 if](https://en.wikipedia.org/wiki/%3F:)
# 所以即使是基本条件，你也需要使用完整的 `if` 语句。
```
