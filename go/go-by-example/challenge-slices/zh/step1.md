# 切片

此挑战中要解决的问题是在 Go 语言中创建和操作切片。你需要创建一个长度不为零的空切片，在切片中设置和获取值，使用 `len` 函数获取切片的长度，使用 `append` 函数向切片中添加新值，使用 `copy` 函数复制切片，以及使用切片运算符从现有切片中获取元素切片。

## 要求

要完成此挑战，你需要对 Go 语言的语法和切片数据类型有基本的了解。你还需要熟悉 `make`、`append` 和 `copy` 函数，以及切片运算符。

## 示例

```sh
# 请注意，虽然切片与数组是不同的类型，
# 但 `fmt.Println` 对它们的呈现方式类似。
$ go run slices.go
emp: [ ]
set: [a b c]
get: c
len: 3
apd: [a b c d e f]
cpy: [a b c d e f]
sl1: [c d e]
sl2: [a b c d e]
sl3: [c d e f]
dcl: [g h i]
2d: [[0] [1 2] [2 3 4]]
```
