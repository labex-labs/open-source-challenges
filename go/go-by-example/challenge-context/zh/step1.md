# 上下文

`hello` 函数通过等待几秒钟再向客户端发送回复，来模拟服务器正在进行的一些工作。在工作过程中，留意上下文的 `Done()` 通道，以获取我们应该取消工作并尽快返回的信号。

## 要求

- Go 语言版本 1.13 或更高。

## 示例

```sh
# 在后台运行服务器。
$ go run context-in-http-servers.go &

# 模拟对 `/hello` 的客户端请求，在启动后不久按下
# Ctrl+C 以发出取消信号。
$ curl localhost:8090/hello
服务器: hello 处理器已启动
^C
服务器: 上下文已取消
服务器: hello 处理器已结束
```
