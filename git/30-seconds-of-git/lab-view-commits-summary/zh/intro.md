Git 是一个强大的版本控制系统，它允许开发者跟踪其代码库随时间所做的更改。Git 最有用的特性之一是能够查看对存储库所做的所有提交的摘要。这在试图了解项目历史或确定何时进行了特定更改时会很有帮助。
