Git 是一个强大的版本控制系统，它允许开发者高效地在项目上进行协作。Git 的特性之一是快进合并，它允许 Git 自动合并没有分歧提交的分支。然而，这个特性有时会导致问题，特别是在处理有多个贡献者的大型项目时。在这个实验中，你将学习如何默认禁用快进合并。
