在 Git 中，子模块是仓库中的仓库。它们允许你将其他仓库作为依赖项包含在你的项目中。当你克隆一个包含子模块的 Git 仓库时，子模块不会自动被克隆。这在尝试构建或运行项目时可能会导致问题。在这个挑战中，你将学习如何克隆缺失的子模块并检出提交。
