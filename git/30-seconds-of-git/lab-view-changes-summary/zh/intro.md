Git 是一种流行的版本控制系统，它允许开发人员随时间跟踪对其代码库所做的更改。Git 最有用的功能之一是能够查看两个提交之间的更改。当试图了解对代码库做了哪些更改，或者试图确定错误原因时，这会很有帮助。
