# 查看最后一次提交

你正在与一组开发者共同处理一个项目，并且你需要查看对该项目的 Git 存储库所做的最后一次提交。你想要查看提交的详细信息，包括提交消息、作者和日期。

## 任务

要查看对 Git 存储库所做的最后一次提交，请按以下步骤操作：

1. 打开你电脑上的终端。
2. 导航到 Git 存储库所在的目录。
3. 查看最后一次提交。

输出将显示最后一次提交的详细信息，包括提交消息、作者和日期：

```shell
commit d22f46ba8c2d4e07d773c5126e9c803933eb5898 (HEAD -> master, origin/master, origin/feature-branch, origin/HEAD)
Author: Hang <huhuhang@users.noreply.github.com>
Date:   Wed Apr 26 14:16:25 2023 +0800

    Added file2.txt
```
