在 Git 中，修正提交（fixup commit）是一种特殊类型的提交，用于修正先前的提交。当你想要对已经提交的内容进行小的更改，而无需创建新的提交时，通常会使用它。当你在一个有许多贡献者的大型项目中工作时，修正提交特别有用，因为它们允许你进行小的更改而不会干扰其他人的工作。
