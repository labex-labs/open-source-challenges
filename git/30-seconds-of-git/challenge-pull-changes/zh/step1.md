# 从远程仓库拉取最新更改

你正在与一组开发者共同处理一个项目，并且你需要确保你本地的代码库副本与团队成员所做的最新更改保持同步。要做到这一点，你需要从远程仓库拉取最新更改。

## 任务

对于这个挑战，我们将使用名为 `https://github.com/labex-labs/git-playground` 的 Git 仓库。

1. 切换到克隆仓库的目录。
2. 从远程仓库的 `master` 分支拉取最新更改。

完成挑战后，你应该会看到一条消息，表明你本地的仓库副本与远程仓库是最新的。
