在 Git 中，子模块是作为子目录包含在另一个仓库中的仓库。子模块允许你在一个更大的项目中为特定项目保留一个单独的仓库。但是，可能会有需要从仓库中删除子模块的情况。在这个挑战中，你将学习如何从 Git 仓库中删除子模块。
