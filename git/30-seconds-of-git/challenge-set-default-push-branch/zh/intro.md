在使用 Git 时，将本地更改推送到远程仓库是很常见的操作。默认情况下，Git 会将你的更改推送到与本地分支同名的分支。然而，有时你可能希望将更改推送到不同的分支。在这个挑战中，你将学习如何将默认推送分支名称设置为当前本地分支的名称。
