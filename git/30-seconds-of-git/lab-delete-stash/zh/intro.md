在 Git 中，贮藏（stash）是一种保存尚未准备好提交的更改的方法。当你需要切换到不同的分支或处理不同的任务，但又不想提交更改时，贮藏就很有用。然而，有时你可能需要删除不再需要的贮藏。在本实验中，你将学习如何在 Git 中删除特定的贮藏。
