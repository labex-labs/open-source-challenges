Git 是开发者用于管理代码库的一种流行版本控制系统。Git 的一项基本特性是能够创建提交（commit），即特定时间点代码的快照。默认情况下，Git 使用用户的姓名和电子邮件来标识提交的作者。然而，在某些情况下，你可能需要以不同作者的身份创建提交。在这个挑战中，你将学习如何在 Git 中以不同作者的身份创建提交。
