Git 是一个版本控制系统，它允许开发者跟踪其代码库中的更改。有时，开发者可能会对代码进行更改，但之后又决定不保留这些更改。在这种情况下，有必要丢弃这些更改并恢复到代码的上一个提交版本。本挑战将专注于在 Git 中丢弃未提交的更改。
