Git 是一种流行的版本控制系统，它允许开发者跟踪其代码库中的更改。Git 的一个实用功能是创建分支的能力，分支本质上是代码库的独立副本，可以独立修改。然而，有时很难跟踪哪些分支包含特定的提交。在这个挑战中，你将学习如何找到不包含特定提交的分支。
